Lab Note B6-003
Chance Operations · 04 — Setup

The rig — where the dice plug in

Same five boxes and one pair of headphones as every Lab Note, plus a physical die, a coin, and a copy of VCV Rack. The new question this page answers is which path each chance device takes into the music — and the iron rule that chance always rides the control path, never the audio path. The audio is what was decided; the control is where the coin is allowed to act.

The rig

Ableton Live 12 Suite

  • Records, edits, arranges; the only MIDI clock master
  • Per-note chance and clip follow actions as native chance devices
  • Max for Live for routing random CV if you skip VCV

Elektron Digitakt II

  • Trig conditions: % probability, A:B ratios, NEI logic
  • Conditional sample locks — a coin choosing a slot
  • The main on-box chance engine for this build

Arturia MicroFreak

  • Source of the bass, drone, and texture audio
  • Receives chance-chosen notes over MIDI from VCV or Live
  • Spice & Dice as a built-in bounded randomizer

VCV Rack (free)

  • Bernoulli Gate — a literal coin in the control voltage
  • Random / sample-and-hold modules for n-way picks
  • Sends MIDI notes or gates to the MicroFreak and Digitakt

MOTU M4

  • 4-in / 4-out interface; the single audio device
  • MicroFreak mono into In 1, Digitakt stereo into 3/4
  • The clock anchor for the whole rig

A die, a coin, paper

  • The cheapest, most honest chance device you own
  • A roll log on paper — the score of what chance decided
  • Use them when the procedure is structural, not per-step

The two paths — and which one chance is allowed on

Audio carries the sound you already committed to. Control carries the decisions still being made. Keep them mentally separate: a chance device that ends up in the audio path is just a fader nobody is holding.

Audio path — never random

MicroFreak Out → TS → M4 In 1 + Digitakt Main L/R → 2 × TRS → M4 In 3/4 → USB-C → Ableton → DT 990 Pro

Control path — where the coin lives

Die / coin / VCV / code → a pick → Boundary you authored → MIDI / CV / a logged choice → Digitakt trig · MicroFreak note · clip order
The iron rule

Chance touches only the control path. It can choose which note, which sample, which section variant, or whether a non-essential trig fires. It must never decide whether the kick exists, what key you're in, or where the peak lands — those are decided, recorded, fixed.

VCV Bernoulli gate — the canonical coin

Patch: a coin that chooses which voice triggers, never whether a voice exists
ClockFrom Ableton via a MIDI-to-CV clock, or VCV's own clock locked to Link. One clock master, as always.
Bernoulli Gate — inputThe steady gate stream you want to route, not create
Bernoulli Gate — probabilityStart at 0.5 (a fair coin); bias toward the safer outcome as you tighten the fence
Output A / Output BTo two different sample triggers or two MIDI note CVs — both musically valid on their own
ModeToggle off for an independent coin each clock; toggle on for a latching A/B with memory
  • Why route, not generate: if the Bernoulli gate decides whether a gate passes at all, you've handed existence to chance. Feed it a guaranteed gate and let it only choose the destination — that keeps the rhythm intact and the chance bounded.
  • Quantize before the synth: always pass random CV through a quantizer set to your scale and your allowed degrees. An unquantized random voltage is the open-radio failure mode in miniature.

Recording discipline for chance takes

Chance makes one new demand on tracking
Record longA probability trig at 35 % needs many bars before its behaviour is audible. Record 4 minutes minimum, never an 8-bar loop.
Log every rollKeep paper next to the keyboard. Write down what the die/coin chose and at which bar — this is your score, and the reflection page depends on it.
One take = one seedA take is one run of the system. To audition another outcome, re-roll and record again — don't edit the take into a different result.
Monitor discipline unchangedAbleton monitor on Auto, M4 MON off for that channel — exactly as the studio reference describes.
One clock to rule them all

VCV, the Digitakt, and the MicroFreak all follow Ableton's clock. A chance procedure that drifts off the grid is two failures stacked — bad timing and an outcome you can't reproduce. Confirm sync before any roll, just as in the studio reference.
