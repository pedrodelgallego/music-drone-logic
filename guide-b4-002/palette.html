Lab Note B4-002
Saturation · 05 — Saturation palette

A recipe per voice, in dB

Each element gets a documented nonlinear treatment: which stage, which curve, what drive, and the output trim that level-matches it. These are starting points calibrated to the gain ledger on page 04 — feed each stage the input it expects and the drive numbers mean what they say. Copy the discipline; the sounds are yours.

How to read a recipe

Every recipe lists input (the level hitting the curve), curve / drive (the timbre), and output (trimmed to match bypass). Set the input first with a Utility, dial the drive, then trim the output until on and off are the same loudness. Only then decide if you like it.

Kick — hard, odd, baked then bus-glued

Stage A — Digitakt source (irreversible)
Per-track Drive35–45 % on T1. Hard, odd grit welded into the sample. Audition against the bass, not solo.
Sample levelLoud but not clipping the box — let the Drive do the clipping, not the converter.
TuneFundamental to A or E to agree with the bass — saturation on a detuned kick smears the low end.
Stage A — Ableton channel on DT KICK
Saturator — TypeAnalog Clip
Input (Utility before)~ -8 dBFS avg
Drive+4 dB static; automated to +9 dB at the peak
Soft Clip toggleOn — rounds the hardest corner, guards against aliasing on the click
OutputTrim -4 dB to match bypass
ProtectEQ HP 25–30 Hz before the Saturator so sub doesn't dominate the curve

Hats & perc — bite without harshness

Stage A — channel on DT HATS/PERC
Digitakt Drive20–30 % — present, not fizzy
Saturator — TypeSoft Sine — gentle odd, keeps the metallic top musical
InputHP 500 Hz before the curve — hats carry no useful low end to distort
Drive+3 dB, Dry/Wet 60 % (parallel — keep the transient clean)
OutputTrim -2 dB to match
WatchThe DT 990's 7–10 kHz lift exaggerates this; if it sounds harsh here, it is probably fine elsewhere — confirm before cutting

Drum bus — shared density

Stage B — Roar on the DRUMS group
ModeSingle band to start; later split so only the 150 Hz – 4 kHz mids drive, sub band clean
Input~ -10 dBFS on the group — set with the group fader, not the channels
Amount / Drive15 % static; automated to 35 % across the build
Color / ToneSlightly dark — tame the added top before it stacks with the hats
OutputLevel-match to bypass; confirm the kick transient still pokes through
Alternative — Drum Buss on the DRUMS group
Drive15–25 %
Crunch10 % — adds high grit; small amounts only
Transients+10 % — claws the kick attack back from the drive
BoomOff or < 10 %, tuned to the kick — do not let it muddy the bass's band

Bass — even body, then rising odd grit

Stage 0 — hardware (irreversible, even)
MicroFreak filter driveOscillator level high, resonance ~ 11 o'clock — push the filter into analog grit. Recorded into the take.
Why evenThe filter's asymmetric overdrive leans even — fattens the bass before any plugin
Stage A — channel on MF BASS
1 — EQ HP40 Hz before the curve — protect the sub from the harmonic generation
2 — Dynamic TubeBias for even harmonics, Drive low — body, not bite
3 — Saturator — TypeMedium (rising odd)
3 — Drive+2 dB intro → automated to +10 dB at the peak — this lane carries the bass's arc
3 — OutputTrim to match bypass at every drive value (automate output inversely if needed)
4 — Sidechain compAfter the saturation, keyed to the kick — so the grit ducks too and never masks the kick
Order is the recipe here

HP → Tube (even) → Saturator (odd) → sidechain is not interchangeable. Move the HP after the Saturator and the sub distorts; move the sidechain before the saturation and the grit stops ducking. The slot is the sound.

Drone — protected, one gentle even stage

Stage A — channel on MF DRONE (kept mostly clean)
1 — EQ HP120 Hz — the drone owns the mids, never the sub
2 — Dynamic TubeDrive low, even bias — adds only body and a hint of air
Drive ceilingNever automated up — the drone is the element deliberately left near-linear
OutputMatch bypass; if you can clearly hear grit, you went too far — back it off
ReverbSits after, fully clean — never send a distorted drone into a long tail or the tail smears

Atmosphere — character distortion, used once

Stage A — channel on ATMOS
Amp / CabinetOne extreme nonlinear pass for texture — a resampled fragment crushed beyond recognition
InputHP 220 Hz first — keep the crush out of the low end
Dry/Wet40–60 % — it is colour, not a lead
OutputSit it around -20 dB on the fader; if you can name what it does, it's too loud

Master — cohesion only

Stage C — master glue (never sound-creation)
Saturator — TypeSoft Sine, Soft Clip on
InputMix peaks ≤ -6 dBFS — a clean, headroomed input
Drive+1 to +2 dB only — a hair of harmonics to seat the mix together
OutputFinal peak ≤ -3 dBFS; no limiter in this exercise
Bypass testToggle it at equal loudness — if you can't reliably hear it, it is doing its job

Palette decision check
  • Does every recipe have a written input, a curve, and a level-matched output?
  • Is the sub protected on every voice — HP before the curve, nothing low into a hard stage?
  • Which voices carry the arc (kick, bass, drum bus) and which stay fixed (drone)? Is that contrast deliberate?
  • If I bypass every saturation stage at equal loudness, does the track still hold? It should be thinner, not broken — proof the saturation is shaping, not propping.
