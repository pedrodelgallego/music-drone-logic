Lab Note B7-001
Parameter Riding · 05 — Ride map

The four-control ride map

This is the reference card. Four controls, each with a target device, a Min/Max range, a job, and a safe zone. Print it, tape it next to the rig, and learn it cold. The build on the next page assumes this map exactly — every gesture refers back to one of these four.

The map at a glance

The whole performance surface, on one card
Encoder 1 — BASS FLTBass Auto Filter Cutoff · 600 Hz → 3.5 kHz · the main arc
Encoder 2 — FEEDBKEcho (Return B) Feedback · 35 % → 70 % · the tension valve
Encoder 3 — VERBDrone reverb send · -12 dB → -3 dB · depth & scale
Pads — MUTESKick / hat / perc / texture · binary, 1-bar quantised · section edits
Read the map this way

Three encoders you turn slowly and a bank of pads you tap on the bar. If you can recite what each one does and where its safe zone ends, you are ready to rehearse. If you cannot, do not record a take yet.

Ride 01 — the bass filter

Encoder 1

BASS FLT — your main arc

Bass Auto Filter Cutoff
TargetAuto Filter (low-pass, 24 dB) on MF BASS → Cutoff
RangeMin 600 Hz · Max 3.5 kHz
JobThe energy arc. Closed = underwater intro; open = the track's brightest moment.
Safe zoneEntire range is musical by design — the danger is speed, not value
How to rideLong S-curves of 16–48 bars. This is the instrument; everything else accompanies it.
Resting stateStart at 600 Hz (closed). Return near there to close the set or loop the take.

Ride 02 — echo feedback

Encoder 2

FEEDBK — the tension valve

Echo (Return B) Feedback
TargetEcho device on Return B → Feedback
RangeMin 35 % · Max 70 %
JobTransitions. Push for 4–8 bars so echoes pile, then pull back as the next section lands.
Safe zoneBelow ~60 % is comfortable; 60–70 % is the near-runaway edge — visit it, never park there
How to rideThe one faster gesture you allow yourself — still spread over bars, never a flick
Resting state35 % floor. Never start or leave a section with this cracked open.
Most dangerous control

Treat the top of this range as a near-feedback control. The Max of 70 % is baked into the map so a panic turn cannot reach runaway — but your judgement still has to bring it back down before the wall arrives.

Ride 03 — reverb send

Encoder 3

VERB — depth and scale

Drone reverb send (to Return A)
TargetMF DRONE → Send A (Hybrid Reverb)
RangeMin -12 dB · Max -3 dB
JobHow large the room feels. Blooms at the peak, recedes at the strip-back.
Safe zoneWhole range is safe; above -3 dB (clamped) the drone would wash the mix out
How to rideMove it with the bass filter — both reach their bright/wet peak together as one swell
Resting state-12 dB (dry). Space is earned later, not the opening state.

Ride 04 — the mutes

Pads

MUTES — section edits, performed live

Drum track mutes (Launchpad pads or Digitakt mute mode)
TargetKick / hat / perc / texture mutes on DT DRUMS sub-lanes or the Digitakt
RangeBinary on/off — launch quantised to 1 Bar
JobArrangement performed live: drop the hat, strip to the kick, bring perc back.
Safe zoneNever mute the kick mid-floor without a plan to return it — the room feels a dropped kick instantly
How to playPrepare the mute before the bar line; let the quantise land it clean
Resting stateKick always on. Hat and perc are the toys; texture is the rare spice.

What stays off the map

Fixed — never under a performance hand
Master outputLives on Live's master or the venue mixer. Only touched to recover.
Kick level & tuningFrozen in rehearsal. A ridden kick is a wrong kick waiting to happen.
MicroFreak / Digitakt internal modulationLFOs and p-locks run themselves — delegated movement you never think about
Bass & drone tuningDecided in the studio; live re-tuning sours the held chord
The pairing rule

Encoders 1 and 3 move together — filter opens as reverb blooms, one swell. Encoder 2 is the punctuation between sections. The pads are the structure. Four moves, three of them slow, all of them named.

  • Each ride encoder is labeled on the EC4 display and on a tape strip on the surface.
  • Every continuous map has its Min/Max set to a musical range — no encoder reaches a value that hurts.
  • Mutes are quantised to the bar so a section edit lands clean.
  • Nothing dangerous (master, kick) is under a performance hand.
  • You can name, from memory, what every mapped control does and where its safe zone ends.
