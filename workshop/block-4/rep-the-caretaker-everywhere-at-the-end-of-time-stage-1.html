Replicate: The Caretaker — Everywhere at the End of Time — Stage 1

Copy the reference's working mechanism, not its style. Build a short original study with Live 12, Digitakt II, MicroFreak, and the MOTU M4, then A/B it against the reference at matched loudness.

The point

This drill teaches one thing: a transformation chain: resonance, saturation, grain, slice, or generation loss. The reference gives you the clue; the build proves whether you understood it.

Before you start
  • Live 12 set saved as REP-THE-CARETAKER-EVERYWHERE-AT-THE-.
  • Digitakt II connected by USB for sync and by main outs to MOTU M4 inputs 3/4.
  • MicroFreak output connected to MOTU M4 input 1 when the lesson needs a synth source.
  • MOTU M4 headphones at the Lab 001 mark; Live tracks set to Monitor Off while recording hardware.
  • Reference on a Live audio track named REF, with Utility inserted for loudness matching.
  • What you will make: a 60-90 second original study where one source is changed by a traceable processor path.
  • What you will turn off: extra layers, decorative effects, and level jumps that hide what the processor is doing.
  • What you will prove: the same transformation behavior remains audible after you change the source.
  • What counts as success: remaining differences are source choices or rig limits, not a missing mechanism.
Reference pass · 10 minutes

Listen before you build

Listen to one clear section of The Caretaker — Everywhere at the End of Time — Stage 1. The opening stage of a six-stage portrait of dementia: 78-era ballroom recordings, still nearly intact, with the degradation system that will consume them only beginning to show — from the artist whose An Empty Bliss Beyond This World anchored lab 007's sampling lesson. Write one short answer for each question before you touch the gear.

  1. What is the smallest repeated behavior you can point to?
    Answer

    Name the behavior, not the mood: timing rule, processor path, space path, decision rule, performance move, or authoring constraint.

  2. Which device or path could create that behavior in your studio?
    Answer

    Pick one place: Digitakt II pattern data, MicroFreak source tone, Live device chain, MOTU M4 input path, or your hands during the print.

  3. What should stay boring so the mechanism remains audible?
    Answer

    Keep supporting material plain so the mechanism can be heard without explanation.

  4. What would prove you copied the system rather than the style?
    Answer

    Use different material while producing the same kind of behavior.

Steps

  • Button map for the steps below: use these controls whenever the lesson names the device.
  • Live 12: ⌘ T creates an audio track, ⇧⌘ T creates a MIDI track, ⌘ R renames the selected track, ⌘ , opens Settings, ⌘ M enters MIDI Map mode, and ⌘ J consolidates the selected clip.
  • Digitakt II: [TRK] + [TRIG 1-16] selects the active track. Press [RECORD] for grid recording and numbered [TRIG] keys to place or remove hits.
  • Digitakt II sound and timing: press [SRC], [FLTR], [AMP], [LFO], [FX], and [TEMPO] for the named pages.
  • MicroFreak: use Preset, Type, Cutoff, Resonance, Matrix, and Save only after the study behaves correctly.
  • MOTU M4: MicroFreak usually enters input 1. Digitakt II main L/R enter inputs 3/4. Inputs 3/4 have no gain knobs, so set their level from Digitakt II Main Volume.

Phase 1 · ~20 min

Put the reference in Live and write the hypothesis

Do this
  1. Drop the reference onto REF and loop the clearest 30-60 seconds.
  2. Add Utility to REF and set a comfortable level.
  3. Write: “The mechanism is ___.” Use the listening tag if needed: decay with a narrative.
  4. Create tracks named SOURCE, SYSTEM, and PRINT.
  1. Work in Live 12 only for this phase: keep REF looped from The Caretaker — Everywhere at the End of Time — Stage 1 and use audio clips, Simpler/Sampler, clip envelopes, and resampling.
  2. Build the phase goal — Put the reference in Live and write the hypothesis — as the smallest Live set that proves a transformation chain: resonance, saturation, grain, slice, or generation loss.
  3. Keep every source on named tracks, then use Utility to match the level against REF before judging it.
  4. Record or resample the result to a fresh audio track named PRINT; do not leave the proof only as devices and MIDI.
  5. A/B ten seconds of PRINT against REF and change only the setting that most affects decay with a narrative.
  1. Use Octatrack MKII for this phase: translate Put the reference in Live and write the hypothesis into Flex/Static machines, slices, recorder buffers, scenes, the crossfader, and main-out recording.
  2. Program the pattern, source, or control path so it demonstrates a transformation chain: resonance, saturation, grain, slice, or generation loss, not just the surface of The Caretaker — Everywhere at the End of Time — Stage 1.
  3. Keep one clear performance control for the main change in this phase; name it before recording so the move is repeatable.
  4. Record the device into Live or a recorder as a full pass, then compare it with REF at matched level.
  5. If the target box suggests an easier shortcut, use it only when the same mechanism stays audible.
Phase 2 · ~35 min

Build the smallest working version

Do this
  1. Choose one plain source: saw, square, noise, or simple wavetable.
  2. Connect MicroFreak to input 1 and set level for peaks near -12 dBFS.
  3. Record the dry source on a track named SOURCE.
  4. Duplicate it to CHAIN and add the one processor that matches the reference: filter, Saturator, Roar, Grain Delay, Beat Repeat, or tape-style loss.
  5. Put Utility before and after the processor. Pre level drives; post level matches loudness.
  6. Print dry and processed versions so the processor path can be A/B checked.
