Hardware Modular & 0-Coast

Physical patching adds something software can't: every connection is visible and tactile, and patch state is a physical object you can photograph. But modular is also the most expensive failure surface on the buy list. This page stays locked until the gate passes.

Optional — gated

You do not own this gear yet. The free substitute is VCV Rack, already used in Lab 024b — use it until the physical patching proves itself as a real need. Buy or build only after this gear passes the Gear Purchase Gate. Come back to this page then.

When and why

Reach for hardware modular when VCV Rack has proved you want the physical constraints: one cable = one signal, no undo, patch memory only by photo. The 0-Coast is the right first step — it is semi-modular (no rack required, self-contained voices) and covers the core west-coast ideas before you commit to a full system.

Key concepts

Signal types

If these four are new to you, read the CV, Gate & Voltage Control reference first — it teaches the concept in full, with a free VCV Rack practice path, before you spend anything on hardware.

CV (control voltage)A slowly moving voltage that controls a parameter — pitch, filter cutoff, panning. Think of it as a knob moved automatically by electricity.
GateA voltage that is either fully on or fully off. It tells a module "start now / stop now."
TriggerA very short pulse — just enough to poke a module into action. Use it where you only need the moment, not the duration.
AudioFast oscillating voltage — the same connectors carry it, but at frequencies you hear. Treat audio and CV as separate paths even when the hardware doesn't force you to.

Utilities matter more than oscillators

New modular users buy voices. Experienced ones buy utilities. An attenuator scales a signal down. An offset shifts a signal up or down. A mult (multiple) copies a signal to several destinations. A mixer adds signals together. A clock divider takes your main clock and outputs a slower version — half speed, quarter speed, and so on. Without these, signals collide at the wrong levels and modulation is uncontrollable.

Clock and reset distribution

Every clocked module must share the same master clock — typically sent from the Digitakt II or Live. Run clock out first; then distribute it through a mult or a dedicated clock module. Add a reset signal on bar 1 so all modules restart together. Without a reset, every power-on starts modules at random points in their cycles.

Patch memory by photo and diagram

Hardware modular has no preset recall. Before you unplug anything, photograph the full patch from above and draw a quick signal-flow diagram. Name the photo with the session date and a one-line description. File it alongside the session recording. That photo is your only way back.

Safe gain levels between modular and your rig

Modular audio runs at a higher level than typical line level — it can easily clip your interface or mixer input. Insert an attenuator or a line-level converter between the modular output and the M4 input. Start with the input gain at minimum and bring it up slowly while watching the meter. Target −12 dBFS on peaks.

0-Coast as a first semi-modular

The 0-Coast accepts MIDI over its 3.5 mm TRS input. Connect it from the MicroFreak or the M4 MIDI out. It has one oscillator with an overtone control (adds harmonics by waveshaping), a slope/contour section (a single envelope that acts like both attack and decay), and a dynamics/LPG (low-pass gate) section.

A low-pass gate (LPG) ties brightness and volume together so that a sound gets duller as it gets quieter — the opposite of a standard filter + VCA setup. This is the defining west-coast character: sounds bloom and then die away rather than cutting off sharply.

The 0-Coast also has a clock input and a random voltage output. Patch the random out into the overtone or contour amount for controlled unpredictability without an extra random module.

To record: route the 0-Coast audio output into the M4, or into a Digitakt II audio input if you want to resample it directly. Set the input gain carefully — see the gain note above. Arm a track in Live and record a full take, then print a committed version before unpatching anything.

Proof gate

Before using this gear in a session

Patch one voice, one modulation path (a CV signal controlling something), and one clocked event (a gate or trigger from the master clock). Record the result as a committed print. Then open your Studio Signal Map and update it to show the new audio path and clock connection. You are done when the print exists and the map is current.
