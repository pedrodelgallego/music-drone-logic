Gear Roadmap

What gear arrives at which block, what each piece is for, and how to do every lab without buying anything new. The default rule: buy nothing for a year. Your current rig already covers the whole workshop.

The default rule

Your six owned boxes — Ableton Live 12 Suite, MicroFreak, Digitakt II, MOTU M4, DT 990 Pro headphones, and Launchpad — are enough to complete every lab in this workshop. New gear is optional. It joins only when a real, named problem in a real project can't be solved without it. That test is described on the Gear Purchase Gate page.

Block-by-block gear schedule

The table shows what gear is introduced at each block. "Core rig" means the six boxes you already own. Everything else is optional — the fallback column tells you how to do the same work without it.

The purchase test

Before buying any gear on the "optional" list, run the five-question test on the Gear Purchase Gate page. A short summary of the test:

1 · What problem does it solve?Name a real bottleneck in a current project — not something you imagine needing in the future.
2 · Which path changes?It must update your audio path, control path, clock path, monitoring path, or print path. If you can't draw the change, you don't understand the purchase.
3 · What does it replace or simplify?If it only adds options without removing anything, it fails. More choices is not more music.
4 · What proof will certify it?You commit to a 30-minute integration test that ends with a recorded result.
5 · What new failure does it create?Name the new fault it introduces — extra gain to set, a second clock to sync, a noise source, more latency. If you can't name it, you can't manage it.
Still want it? Wait 30 days.

Spend that month pushing the current rig at the same problem. Most wants evaporate. The real ones get clearer and pass the gate with a concrete answer to question 1. See the full test at Gear Purchase Gate.

Core-rig fallback

Every gear-dependent lab in this workshop has a software path. You can do the whole workshop on the six boxes you already own, plus free software. Nothing is blocked.

Hardware you already own

  • Ableton Live 12 Suite — DAW, sampler, effects, soft synths
  • Arturia MicroFreak — digital oscillators, analogue filter, step sequencer
  • Elektron Digitakt II — drum machine, sampler, MIDI sequencer
  • MOTU M4 — audio interface, four inputs, headphone out
  • Beyerdynamic DT 990 Pro — open-back monitoring headphones
  • Novation Launchpad — session and clip launch controller

Free software to add

  • VCV Rack — free software modular for patching and generative work
  • Pure Data — visual programming for rules and chance operations
  • SuperCollider — code-based sound synthesis and scheduling
  • TidalCycles — livecoding for pattern-based music
  • Scala — tuning and scale editor for microtonal work (Lab 026)
  • Youlean Loudness Meter — LUFS and true-peak metering (free tier)

Tiered budget guide

If you decide to spend, this is the order that adds the most useful capability per dollar and per unit of added complexity. Each tier assumes the previous tier is already in place.

TierSpendWhat you addWhen to consider it
$0 — Current rig Nothing Complete workshop coverage; all labs doable Default. Start here and stay here for at least a year.
Low — Accessories ~$20–80 Contact mic, induction coil (e.g. LOM Usi, Elektrosluch) — small, no setup cost, no new signal path After Block 2, when field and electromagnetic capture becomes a recurring source practice
Medium — One addition ~$150–350 Portable recorder (Zoom H5/H6) or one effects pedal (delay or reverb in a send/return loop) After Block 2 or Block 5 respectively; only if the lab's fallback path has already proved the need
High — Major instrument ~$400–900 Semi-modular synthesiser (0-Coast, Moog Grandmother) or analogue poly synth — a new voice type Only after VCV Rack or software polyphony work has proved the recurring need; must pass the full Purchase Gate
Ongoing — Expandable Open-ended Eurorack modular — bought one module at a time, only when VCV Rack proves a specific function fixed gear can't do Last priority; only after completing the full workshop once
  • Buy nothing for a year. This is not a budget constraint — it is a practice constraint. The rig you own is already complex enough to spend a year learning deeply. Every new box resets some of that learning.
  • When a purchase is justified, it earns a page. Any gear that passes the Purchase Gate gets an integration lesson — routing, clock, gain, monitoring, print, and failure mode — before it is used in a lab. Until that page is written, the gear stays off the bench.
  • The Digitone II exception. The Digitone II has already passed the Purchase Gate: it answers question 1 (FM synthesis and polyphony are a genuine gap in the current rig) and has its own integration page at Block 0.
