Study Roadmap

One path, four horizons

Read this top to bottom — it's the order you actually work in. The path moves through four horizons, not calendar years: you cross into the next when you can do the current one, at your own pace. Each block lists its labs; the optional side quests are deep-dive Lab Notes you can take whenever you want the long version.

The working loop

Every lesson turns listening into a test. Say what you think is happening, wire it up, make a result, compare it with the original, then write down the rule you learned.

Listen→ Say what is happening→ Set up the path→ Set the levels→ Build or measure→ Record the result→ Compare→ Fix the difference→ Write the rule
