Synthesis from Scratch

A synth makes sound in three steps: an oscillator makes a raw tone, a filter carves it, and an amplifier (shaped by an envelope) decides when it's loud. Add an LFO for movement and that's the whole machine. This primer drills each control on its own, then hands you a cookbook of sounds to build from a blank patch.

Primer · the atoms one control at a time · ends in a recipe cookbook + a recreate-by-ear gate

Your anchor instrument

Build every atom on the MicroFreak — it has all of them on the panel: an oscillator with many engines, a real analog filter, an envelope, an LFO, and a mod matrix. One control needs a second tool: pulse width is clearest in Live's Analog (or Operator), so you'll hop there for that single atom.

The whole machine, in one line

Oscillator
raw tone → Filter
carve it → Amplifier
when it's loud → out

The envelope drives the amplifier (and often the filter). The LFO wobbles any of them automatically. Master those four boxes and you can build anything.

Atom 1

Oscillators — the raw tone (and pulse width)

The oscillator makes a repeating wave. Its shape sets the raw character before anything else touches it.

Hear each waveform (MicroFreak)
  1. Pick the Basic waves oscillator type. Hold one note.
  2. Turn the Wave knob through the shapes and listen: sine (pure, hollow), triangle (soft), sawtooth (bright, buzzy — every harmonic), square (hollow, woody).
  3. Rule of thumb: more edges in the wave = more harmonics = more for the filter to carve later.
Pulse width / PWM (hop to Live's Analog)
  1. A square is a pulse that's on half the time. Pulse width changes that ratio — from a fat square to a thin, nasal buzz.
  2. In Live's Analog, set OSC 1 to Pulse. Move the PW (pulse width) knob from 50% toward the extremes and listen to it thin out.
  3. PWM = pulse-width modulation: route an LFO to PW (small amount, slow). The shifting width makes one oscillator sound like two slightly detuned ones — the classic "string" shimmer. This is the atom most synth courses build a whole patch around.
Prove it

Without looking, can you tell a sawtooth from a square by ear? And can you make a static pulse move with PWM? If yes, this atom is done.

Atom 2

The filter — carving the sound

A filter removes part of the sound's frequency range. It is the single most expressive control on a synth.

On the MicroFreak's analog filter
  1. Play a sustained sawtooth. Sweep Cutoff down: the brightness disappears top-first. This is a low-pass filter — it keeps the lows, removes the highs.
  2. Now turn up Resonance. A whistle appears right at the cutoff point — the filter emphasising itself. Sweep the cutoff with resonance up: that vocal "wah" is the sound of a thousand records.
  3. Push resonance near maximum: on many synths the filter starts to sing its own tone (self-oscillation). The MicroFreak's filter goes vocal and ready to scream.
Prove it

Make one held sawtooth sound like three different instruments using only the cutoff and resonance — no notes changed.

Atom 3

The envelope — one stage at a time (A · D · S · R)

The envelope is the shape of a sound over the life of one note. Learn the four stages separately, then together. Route the MicroFreak's envelope to the filter cutoff so you can hear each one.

The four stages — set the others to zero, then move just one
Attackhow fast it starts. Zero = instant click. Long = a slow swell with no clear beginning.
Decayhow fast it falls from the peak after the attack — toward the sustain level.
Sustainthe level it holds at while the key is down. This is a level, not a time.
Releasehow fast it fades after you let go. Zero = abrupt stop. Long = a tail.
Do this
  1. Set attack 0, decay 0, sustain full, release 0 — a plain on/off gate. Play.
  2. Raise attack only: the note now fades in. Return it to 0.
  3. Drop sustain to zero and raise decay: now you get a pluck — it stabs and dies even while held. This decay-to-zero shape is the heart of basses, plucks, and stabs.
  4. Return sustain up, then raise release: the note now rings on after you let go.
Prove it

By ear, point to which stage makes a pad a pad (long attack + release) and which makes a pluck a pluck (zero sustain, short decay).

Atom 4

LFOs — automatic movement (rate · depth · shape)

An LFO is a slow, automatic knob-mover. It makes no sound itself — it wobbles another control on a repeating cycle. Three knobs control it.

On the MicroFreak (route it in the mod matrix)
  1. In the mod matrix, route LFO → Cutoff. You set this connection's strength with the depth (amount): start tiny.
  2. Rate = speed. Slow = a breathing filter (good for drones). Fast = a wobble. Very fast (audio rate) = it changes the timbre itself.
  3. Depth = how far it moves the cutoff. Small = subtle life; large = a full sweep.
  4. Shape = the path it takes: a smooth sine/triangle (gentle), a ramp/saw (rise then snap), a square (jump between two values), or random (sample & hold — a new value each cycle).
  5. Try LFO → pitch at a tiny depth and slow rate: that's vibrato. Bigger depth: a siren.
Prove it

Make a held tone feel "alive" with one slow LFO — then make it "wobble" — then make it "jump" — changing only rate and shape.

The cookbook — build these from a blank patch

Now combine the atoms. Start each from an init/blank patch (one oscillator, filter open, plain on/off envelope) and make only the moves listed. Build them in order — each adds one idea.

Proof gate

You pass this primer when you can…
  • Build the cookbook: make all nine recipes from a blank patch, each in under two minutes.
  • Name the atoms by ear: identify the waveform, hear the filter move, and call which envelope stage is doing the work.
  • Recreate two by ear: have someone (or a preset) play a pluck and a pad; rebuild each from scratch without looking at the source.
