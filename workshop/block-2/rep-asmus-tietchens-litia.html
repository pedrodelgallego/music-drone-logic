Replicate: source erased into pure form

Copy one clear section of Asmus Tietchens — “Litia”. Build a short study where processed sources are unrecognisable and arranged as pure form, and compare it to the reference at the same loudness.

OverviewListeningLabsRebuildCapstoneRep lessonsDashboard
The point

This drill is about one decision: process found or recorded sources until they are completely unrecognisable, then arrange the abstract results as pure form. This is abstract synthetic concrète — the origin of the material is irrelevant and unknowable, and what remains is shape, timbre and structure. Do only the steps that erase the source and foreground the form.

Before you start
  • Gear: any source to start from — a few captures from a past lab, or one synth pass — plus Live's processing chain; DT 990 Pro for judging; the lab-006 capture log repurposed to track each processing stage
  • New set saved as REP-LITIA — no tempo; this is timed in seconds and arranged by shape, not pulse
  • Heavy processing can spike unpredictably — re-check gain after every transform, peaks ≈ −12 dBFS; phones at the lab-001 mark, because abstract material gives no loudness reference
  • Timebox: 90–120 minutes — one source, several processing passes to erase it, one short formal arrangement, then file it and stop
  • What you will make: a short replica of Asmus Tietchens — “Litia”. Focus on this effect: a source erased by processing and arranged as pure abstract form.
  • What you will turn off: any attachment to the original source — recognisability is the enemy here.
  • What you will prove: your version and REF use the same core mechanism: abstract synthetic concrète — unrecognisable processed sources arranged as pure form.
  • What counts as success: remaining differences are which shapes you arranged, not the source still being identifiable.
Reference pass · 10 minutes

Listen before you build

Listen to about one minute of Asmus Tietchens — “Litia”. Choose the clearest short section. Write one short answer for each question before you touch the gear.

  1. Can you name any source?
    Answer

    No — and you are not meant to. Whatever was recorded or generated has been processed past recognition. The ear gives up guessing causes within seconds and starts hearing shape instead.

  2. What do you attend to instead of the source?
    Answer

    Form — the way an event rises and falls, its timbre, how one shape sits against the next. With the cause removed, structure and texture become the entire content.

  3. Does it sound “electronic” or “recorded”?
    Answer

    Neither, exactly — that ambiguity is deliberate. It could be a processed recording or a synth; you cannot tell, and it does not matter. The processing collapses the difference, which is why it is called synthetic concrète.

  4. Why erase the source at all?
    Answer

    To compose with shape alone. Recognisable sources carry associations that distract from form. Erasing them frees the listener — and the composer — to deal in pure structure, austerely and abstractly.

  5. Where is the composition, if you cannot name anything?
    Answer

    In the arrangement of abstract shapes — which events, in what order, at what spacing and proportion. The form is the only thing left, so the form is everything.

Now build the section you described. Do not build from memory.

Steps

  • Button map for the steps below: use these controls whenever the lesson names the device.
  • Live 12: ⌘ T creates an audio track, ⇧⌘ T creates a MIDI track, ⌘ R renames the selected track, ⌘ , opens Settings, ⌘ M enters MIDI Map mode, and ⌘ J consolidates the selected clip.
  • Resampling in Live: set a track's input to Resampling, arm it, and record the processed output back to audio so each transform stage prints to a new clip.
  • Sound target: one 60–90 second study where no source is identifiable and abstract shapes are arranged as the whole content.
  • One starting source, taken through several processing stages until it is unrecognisable.
  • The interest comes from the arrangement of shapes — proportion, order, spacing — not from the source.
  • Processing and arrangement only — if a listener can name the source, you have not gone far enough.

Phase 1 · ≈ 10 min

The reference and the erasure rule

Phase focus: the reference and the erasure rule. Follow the steps in order, then compare against REF before moving on.

Do this
  1. ⌘ T, rename the track REF (⌘ R), drag the reference section in, and loop it at ≈ −12 dBFS with Audio Effects → Utility.
  2. Pick exactly one starting source — a single capture from a past lab, or one synth pass; write what it is in the log (you'll erase it).
  3. Write the rule in full: process the source until no listener could name it, then compose with the abstract results only.
  4. List the transforms you'll chain: pitch shift, extreme time-stretch, ring modulation, heavy filtering, granular smear, resample.
  5. Decide the membership test: if I can still hear what it was, it is not abstract enough — process again.
  1. Work in Live 12 only for this phase: keep REF looped from Asmus Tietchens — “Litia” and use audio clips, effect chains, and resampling.
  2. Build the phase goal — the reference and the erasure rule — as the smallest Live set that proves abstract synthetic concrète — unrecognisable processed sources arranged as pure form.
  3. Keep every stage on named tracks, then use Utility to match the level against REF before judging it.
  4. Record or resample the result to a fresh audio track named PRINT; do not leave the proof only as devices and MIDI.
  5. A/B ten seconds of PRINT against REF and change only the setting that most affects whether the source is erased.
  1. Use Octatrack MKII for this phase: load one source into a Flex machine, then translate the reference and the erasure rule using pitch, time-stretch, the LFOs, scenes, the crossfader, and main-out recording to erase it.
  2. Program the control path so it demonstrates abstract synthetic concrète — unrecognisable processed sources arranged as pure form, not just the surface of Asmus Tietchens — “Litia”.
  3. Keep one clear performance control for the main move in this phase; name it before recording so the move is repeatable.
  4. Record the device into Live or a recorder as a full pass, then compare it with REF at matched level.
  5. If the target box suggests an easier shortcut, use it only when the source is erased.
Expected result

The reference loops at ≈ −12 dBFS, one starting source is chosen, and the erasure rule plus a chain of transforms are on paper before processing begins.

Phase 2 · ≈ 35 min

Erase the source — process in stages

Phase focus: erase the source. Follow the steps in order, then compare against REF before moving on.

Do this
  1. Apply the first transform to your source, re-check gain, and resample the result to a new clip named STAGE·1.
  2. Apply the next transform to STAGE·1, resample to STAGE·2, and so on — each stage built on the last.
  3. After each stage, ask: can I still tell what this was? If yes, keep processing; if no, bank it as an abstract shape.
  4. Aim to bank 5–8 distinct abstract shapes — short events, drones, swells, grains — all unrecognisable.
  5. Log each stage's transform in the repurposed lab-006 log so the chain is repeatable.
  6. Discard any clip where the source still leaks through; only fully erased shapes go forward.
  1. Work in Live 12 only for this phase: keep REF looped from Asmus Tietchens — “Litia” and use effect chains, clip envelopes, and resampling.
  2. Build the phase goal — erase the source — as the smallest Live set that proves abstract synthetic concrète — unrecognisable processed sources arranged as pure form.
  3. Keep every stage on named tracks, then use Utility to match the level against REF before judging it.
  4. Record or resample each stage to a fresh audio track named PRINT; do not leave the proof only as devices and MIDI.
  5. A/B ten seconds of PRINT against REF and change only the setting that most affects whether the source is erased.

Phase 3 · ≈ 30 min

The study — arrange shape as form

Phase focus: the study. Follow the steps in order, then compare against REF before moving on.

Do this
  1. Lay your abstract shapes in Arrangement and build one 60–90 second study by proportion and order alone.
  2. Compose with form: a long shape against short events, a swell answered by silence, a dense passage giving way to a sparse one.
  3. Pay attention to spacing and proportion — these are now your only tools, so make them deliberate.
  4. Keep it austere; resist adding warmth or melody to make it “nicer” — the abstraction is the aesthetic.
  5. Place at least one moment of silence as a structural event, not just a gap.
  6. Print the study to one audio track named PRINT.
  1. Work in Live 12 only for this phase: keep REF looped from Asmus Tietchens — “Litia” and use the banked shapes plus level and clip arrangement.
  2. Build the phase goal — the study — as the smallest Live set that proves abstract synthetic concrète — unrecognisable processed sources arranged as pure form.
  3. Keep every shape on named tracks, then use Utility to match the level against REF before judging it.
  4. Record or resample the result to a fresh audio track named PRINT; do not leave the proof only as devices and MIDI.
  5. A/B ten seconds of PRINT against REF and change only the setting that most affects whether the form reads clearly.
  1. Use Octatrack MKII for this phase: translate the study into Flex/Static machines, slices, recorder buffers, scenes, the crossfader, and main-out recording.

If it fails

Verify against the record

Solo-flip between REF and your study at matched loudness — phones at the mark, no level riding while you judge.

  • Can a listener name any source?
    Answer

    That is the pass condition, inverted: they should not be able to. Play it to fresh ears and ask “what is this made of?” A working study gets a shrug — they hear shapes, not causes. If they say “that's a voice” or “that's a piano,” process that shape further. The record passes by giving the ear nothing to identify.

  • Does the form hold attention on its own?
    Answer

    With the source gone, only structure is left to carry the piece — so the test is whether the arrangement is interesting as pure form. If it feels flat or random, the proportion and order need work, not more processing. Re-sequence the shapes until the rises, falls and silences feel inevitable, the way the reference's austere structure does.
