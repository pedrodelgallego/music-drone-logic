Resonance · 04 — Setup

The rig for a filter ride

The same five boxes as Lab Note B1-001, wired the same way — but this topic puts unusual demands on gain-staging, because a resonant peak is loud and narrow. This page is the wiring and metering reference the build assumes: where the filter lives, what feeds it, and how to keep an 18 dB peak from wrecking the chain.

Three filters, three voices

You have three usable resonant filters. They are not interchangeable — each sings differently and is driven differently. Decide on the decisions page which is primary; this build uses the MicroFreak filter as the lead and the Ableton Auto Filter as the long-form arc.

MicroFreak filter — key-tracking the analog state-variable
Resonance knobPast ~3 o'clock the 12 dB state-variable filter self-oscillates and produces a sine
Keyboard → cutoffIn the filter section enable keyboard tracking so the resonant pitch follows the note played
Excite itA short, bright oscillator burst (Wavetable at high Wave, fast envelope) "plinks" the resonance — bell tones from one note
Cycling envelope → cutoffIn loop mode, a slow cycling envelope makes the self-oscillated pitch glide between two values forever
Ableton Auto Filter — the filter as the slow arc
Filter typeClean self-oscillates politely; the analog-style circuits (OSR / PRD / SVF) add their own grit as Q climbs — pick by ear
ResonancePush toward maximum until a steady tone appears with no input — that is self-oscillation engaged
FrequencyThis is now your pitch knob. Map it to a controller or draw it on a scale-quantized lane
Envelope amountSmall positive envelope → each note "blooms" the resonance upward, a filtered pluck
LFO → FrequencySlow, small depth = the resonant tone drifts like a theremin; large depth = siren
Digitakt II per-track filter — resonance as rhythm
FLTR — ResonanceRaise per track; on a noise or texture sample the resonant peak gains pitch as on any source
P-lock the cutoffHold a trig, turn FLTR Freq — the resonant pitch changes step by step, a filter melody locked to the sequence
FLTR envelopePer-step envelope depth turns a flat sample into a resonant pluck only on the trigs you choose
Sound lockLock different cutoff values to give one drum track a sung two- or three-note line from its own filter

Signal path — where the resonance happens

Lead voice — MicroFreak filter as the instrument

MicroFreak osc excites → analog filter (high Q) → TS → M4 In 1 → MF FILTER track peak limiter → Ableton

Arc voice — Auto Filter on a recorded flat source

dry source clip → Auto Filter (cutoff = pitch) → peak-tame Utility / limiter → saturation → master
Recommended path

Use the MOTU M4 as the single audio device: MicroFreak mono into input 1, Digitakt II stereo into inputs 3/4, headphones from the M4. Ableton is the only MIDI clock master; Digitakt II and MicroFreak both receive clock and transport over USB. One device, one clock — the same stable base as Lab Note B1-001.

The resonant gain chain — the part that is different

A high-Q peak can add 12–18 dB at one frequency, and it appears the instant you sweep the cutoff through a strong partial. Set the chain so the peak, not the average, has headroom. Meter it once, deliberately, before you commit a take.

Set in this order, watching the peak — not the average
1 — MicroFreak master volume~ 60–70 %, lower than usual. Hold a note and slowly sweep the cutoff across its full range while watching the next stage. The loudest moment is your reference, not a steady note.
2 — M4 Input 1 gainSet so the loudest resonant peak hits healthy amber, never red. If a normal note looks quiet, that is correct — you are leaving room for the peak.
3 — Ableton MF FILTER meterPeaks at -12 to -8 dBFS during the sweep. The narrow resonant transient is what you are metering, so trust the peak hold, not your ears.
4 — Peak limiter on the trackA Limiter or fast Utility-gain catch set to a ceiling of -6 dBFS, only catching the rare overshoot. This is a safety net, not a sound — if it is working hard, fix stage 1.
5 — Phones knobComfortable, then leave it. A resonant whistle is fatiguing; protect your ears and your judgement by not riding this knob.
The golden rule, restated for resonance

Fix the peak at the earliest stage that causes it. If the resonant ping clips, pull the MicroFreak volume down — do not pull the track fader, which leaves the converter still clipping. The peak you can't hear (it is brief) is the peak that ruins a take.

  • Resonance + long reverb = whistling artifacts. A high-Q peak fed into a long reverb tail can ring at one frequency forever. High-pass the reverb send and keep the send modest on the lead — the build sets this explicitly.
  • Headroom is mandatory, not optional, here. Track peaks at -8 dBFS leave room for the resonance to spike, for saturation to add level, and for the master bus. A take where the resonant ping clips is a take you re-record.
