Lab Note B5-002
Reverb · 04 — Setup

Wiring the room — sends, returns, and the freeze path

The same five-box rig as the rest of the course, set up so reverb is a destination, not an afterthought. Two return tracks carry two depths; one extra return exists only to be frozen and printed. This page is the signal-path reference the build assumes — keep it open in another tab.

The rig, in this context

Ableton Live 12 Suite

  • Holds the reverb: Hybrid Reverb on return tracks
  • Freeze, resample, and automation of the space
  • Master MIDI clock for both hardware boxes

Elektron Digitakt II + Arturia MicroFreak

  • Dry sources: kick, perc, bass, drone — recorded clean into Ableton
  • Digitakt's onboard reverb used sparingly, as a near room only
  • MicroFreak drone is the main thing the big room acts on
The rule that makes this work

Record every source dry. The space is built in Ableton, on returns, where it stays editable. A reverb baked into a take can never be made smaller — only re-recorded.

The reverb signal path

Dry source tracks → send A → A — NEAR ROOM → send B → B — FAR HALL → send C → C — FREEZE → master

Build the three returns

Return A

Near room — placement

Hybrid Reverb settings
A — NEAR ROOM
AlgorithmRoom (Hybrid Reverb's algorithmic Room, or a small Convolution room IR)
Pre-delay15 ms — keeps transients in front
Decay0.8 – 1.2 s
SizeSmall–medium
Dry/Wet100 % — it is a return
  • Insert an EQ Eight before the reverb on the return: HP 300 Hz, LP shelf -3 dB at 9 kHz.
  • This is the room that gives drums and bass a sense of place without any audible tail.
Return B

Far hall — blur

Hybrid Reverb settings
B — FAR HALL
AlgorithmHall (Convolution Hall or large algorithmic Hall)
Pre-delay40 ms — large room, still readable onsets
Decay6 – 8 s
SizeLarge
Dry/Wet100 %
  • EQ Eight before it: HP 350 Hz, LP shelf -4 dB at 7.5 kHz — the far room should be darker than the near one.
  • Add a Compressor after the reverb on this return, sidechained to the kick (Audio From DT DRUMS, Ratio 4:1, ~3 dB reduction) so the wash ducks open between kicks.
  • This is where sounds go to dissolve. The drone lives mostly here.
Return C

Freeze — material to be printed

Hybrid Reverb settings
C — FREEZE
AlgorithmAny large algorithm with a Freeze control
Pre-delay0 ms — full blur is the point here
Decay12 s (irrelevant once frozen)
FreezeOff until you capture; then mapped to a Launchpad pad
Dry/Wet100 %
  • This return exists to be frozen and resampled, not to play under the whole track. Send it a single drone note, hit Freeze, and the tail holds forever.
  • Keep its send fader low in the normal mix; you raise it only during the capture pass.

The freeze-and-print workflow

Technique

Freeze, then commit

Capture the cloud
  • Send a single drone note (or one chord, or a snare tail) to C — FREEZE. While it rings, click Freeze on the Hybrid Reverb — the current tail now sustains indefinitely with no oscillator.
  • Create a new audio track, Audio From → Resampling, Monitor Off. Arm it.
  • ⌘ ⇧ R and record 16–32 bars of the frozen cloud. Un-freeze when done.
  • Drag the recorded clip to the ATMOS track. It is now material — reverse it, stretch it, filter it. The effect has become a layer you can turn off without losing the sound.
  • Hardware version: the Digitakt II's reverb won't freeze, but you can resample a pattern through it with a long decay and high feedback, then keep only the tail — the same idea on the box.

Why bass gets no reverb

Low frequencies carry almost no spatial information, and reverberant low end is the fastest route to mud. The bass stays dry and near; the room is built above it. Send only sounds that benefit from space.

Decision check
  • Are all three returns set to 100 % wet? Check before recording anything.
  • Is every return high-passed, so the room never thickens the low end?
  • Is the far hall ducked to the kick, so the wash breathes instead of masking the groove?
  • Is every source recorded dry, with the space living only on the returns?
