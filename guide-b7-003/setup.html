Lab Note B7-003
Feedback Safety · 04 — Setup

The rig: closing the loop with the ceiling built in

This page builds the instrument as a signal path — and the safety devices are part of that path, not an afterthought. You build the limiter, the capped control, and the panic path before the loop is allowed to oscillate. Two rigs are given: a no-input hardware mixer and an all-software self-oscillating loop. Pick one. Beginners take the software rig — every safety rule is trivial to enforce there.

The signal path, in principle

Whatever the gear, the topology is the same: a source point, a return path, a frequency-shaping stage that decides the tone, a limiter that caps the level, and a kill that opens the loop. Read the flow left to right, then build it.

The return from "Kill / panic" feeds back to "Loop source" — that wrap is what makes it an instrument. The limiter and kill sit inside the loop, before the wrap, so a runaway is caught on the path rather than at the master.

Build order is a safety feature

Insert the limiter and wire the panic path first, while loop gain is still well below unity and nothing can oscillate. Only after both are confirmed working do you raise gain toward the edge. Never build the loud part first and the safety part second.

Rig A — software self-oscillating loop (recommended)

Setup A

Two tracks and a return, wired into a loop

Create the loop with a send and a routed return
  • Make an audio track FB LOOP and a return track FB RETURN.
  • On FB LOOP, set its output to Sends Only and turn the send to FB RETURN up. On FB RETURN, set its output back to FB LOOP's input. That wrap is the loop. (In Ableton this is done with track input/output routing; in any DAW it is "route the return's output to the source track's input".)
  • The loop currently has no source — that is correct. A self-oscillating filter or a tiny noise seed will start it on page 06. Right now you are wiring an empty, silent loop on purpose.
The frequency-shaping stage — what makes it a pitch
  • On FB RETURN, first device: a resonant filter (Auto Filter, band-pass, high resonance) or an EQ with one sharp narrow boost. This is your "Rainforest object" — it decides which frequency oscillates.
  • Set its centre frequency to the pitch you chose on page 03. The loop will sing there because that is where loop gain is highest.
The capped ride control
  • The send amount (or the return track's gain) is the one control that crosses unity. Map it to a single fader or encoder.
  • ⌘ M to MIDI-map it to one pad/fader, then set the mapping's Min/Max so the maximum sits a clear margin below runaway — you will measure that point on page 06 and dial the cap in. The control physically cannot reach the scream.
  • Map this one control and no other. The fewer live hands on a feedback loop, the safer it is.
The in-circuit limiter
  • Last device on FB RETURN, before the wrap back to source: a Limiter, ceiling -3 dBFS. This is the hard ceiling. It is not the master limiter — it lives on the loop.
  • Confirm signal flow: filter → (any tone shaping) → limiter is the last thing the signal hits before returning. If the limiter is not last, a stage after it can re-amplify past the ceiling.
  • Set the ceiling now, in this quiet state, and do not touch it again today.
The panic path
  • Map one dedicated pad to mute FB RETURN — muting the return opens the loop, so it cannot sustain. This is your kill.
  • Better still, map it to a kill that does both: mutes FB RETURN and zeroes the FB LOOP send. One pad, instant silence.
  • Put it in a fixed spot, tape a label on it, and never remap it. You will rehearse hitting it blind on page 06.

Rig B — no-input hardware mixer

Setup B

A small mixer patched into itself

Physical loop
  • Take a small analog mixer. Patch a send/aux output back to a channel input, or a sub-group output back to a channel — the output feeds an input, and the loop is closed.
  • Bring the mixer's main output into your interface, then back out to the speakers through the interface. This puts a software limiter (below) between the mixer and the room.
  • Use the channel EQ on the looped channel as the frequency-shaping stage — boost narrowly to choose the oscillation pitch, exactly as the filter does in Rig A.
The ceiling, kill, and ride for hardware
  • Limiter: on the interface return into the DAW, a Limiter at -3 dBFS before the signal reaches the master/speakers. The analog loop has no built-in ceiling, so the digital limiter is mandatory.
  • Ride control: one channel fader is the encoder that crosses unity. Mark the runaway point on the fader strip with tape (page 06) and treat the tape as the cap — hardware faders cannot be range-limited, so the limit is a line you do not cross plus the limiter behind it.
  • Panic path: the mixer's main mute, or pulling the looped channel fader fully down, in one labeled motion. If the mixer has a mute group, assign the loop channel to it and tape the mute button.
  • Beginners, note: hardware gives no software range-cap, so the limiter and your discipline carry more weight. This is why the software rig is recommended first.

Monitoring rules — non-negotiable

How you listen while building a feedback loop
Speakers onlyBuild and test on monitors at a level you could walk away from. Never in headphones — a jump to the ceiling can injure your ears before your hand moves.
Start lowSet monitor volume lower than feels normal before raising loop gain. You can always turn the room up; you cannot un-hear a scream.
Hand on the killThe first time you raise gain to the edge, your hand is already resting on the panic pad. Always.
House engineerOn stage, confirm the house engineer can pull the master. A second pair of hands is part of the safety system, not a courtesy.
Confirm before page 05

The limiter is the last device on the return at −3 dBFS. The ride control is mapped and range-capped (Rig A) or taped (Rig B). The panic pad is mapped, labeled, and in its fixed spot. Monitoring is on speakers. Only when all four are true do you let the loop oscillate.

Setup troubleshooting

Symptom → check
No sound at all when gain is raisedLoop gain still below unity, or filter centre not where the loop wants to ring. Nudge the filter frequency, raise gain slowly — this is the safe state to be stuck in.
Instant scream the moment gain comes upRide control's range is too wide or the filter resonance is extreme. Lower the cap; widen the filter Q so the approach is gradual.
Loud peak gets past the ceilingA gain stage sits after the limiter. The limiter must be the last device before the wrap. Re-order.
Panic pad does not silence itIt mutes the wrong track, or mutes the source but not the return. Kill must open the loop — mute the return.
Pitch drifts on its ownNormal: loop gain drifts with room and gear. That is why you ride one control and why the panic path exists.
