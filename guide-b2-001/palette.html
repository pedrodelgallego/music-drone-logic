Synthesis from Zero · 05 — Palette

Seven recipes, every parameter spelled out

These are the building blocks of the finished piece — kick, hat, clap, bass, lead, pad, drone. Each is a complete recipe: the method, the signal flow, and a parameter table with sane starting values you can dial without guessing. Read these, then assemble them on the Build page. Every value is a starting point you should be able to defend, not obey.

How to read a recipe

Each recipe names the method, draws the signal flow, and gives a parameter table. Numbers in ms are envelope times; numbers in Hz are frequencies; FM ratios are written carrier : modulator. Build the sound, then change one value and listen — the recipe is a launchpad, not a preset to copy.

1 · Kick — a falling sine plus a click

Method: pitch-enveloped sine (the body) summed with an ultra-short noise/high-sine generator (the click). The punch lives entirely in the first ~10 ms; the weight lives in the body's decay.

sine osc A → pitch env
150 → 50 Hz, ~5 ms → amp env
0 ms atk · 250 ms dec + click osc B
noise, 3 ms dec → kick
Kick — Operator
Oscillator ASine, Coarse 1, level full — the body
Resting noteplay ≈ 50 Hz (F1), tuned to the track root
Pitch env amount+1 to +2 octaves (starts ≈ 150 Hz)
Pitch env time3–8 ms decay — this is the "click → body" snap
Amp env (A)A 0 ms · D 250 ms · S 0 · R 10 ms
Oscillator B (click)Noise or high sine, parallel (not modulating), amp D 2–5 ms, low level
FXNone — a dry kick cuts. Saturator +1–3 dB only if it needs grit.
  • MicroFreak alternative: low note, the single envelope assigned to Pitch with a very short attack/decay for the sweep. The separate click is hard with one envelope — layer a second short take or accept a softer transient.
  • The fault to know: a pitch-env time of 80–150 ms turns the kick into a "boop" — a slow glide with no punch. Keep it short.

2 · Hat — gated high-passed noise

Method: noise shaping. Noise has no pitch — the high-pass chooses the colour, the decay is the instrument. Closed and open hats are the same source at two decay times.

white noise → high-pass
~7–8 kHz → amp env
0 ms atk · 40 ms dec → hat
Hat — Operator/Wavetable noise + filter
SourceWhite noise (Operator Noise waveform or Wavetable noise osc)
FilterHigh-pass, cutoff ≈ 7–8 kHz, mild resonance
Amp env (closed)A 0 ms · D 40 ms · S 0
Amp env (open)A 0 ms · D 300 ms · S 0 — duplicate the patch, lengthen decay
ChokeProgram the closed hat to cut the open hat (open on the off-beat)
Metallic optionTwo square/saw oscillators at inharmonic ratio (≈ 1 : 1.41) instead of noise — FM-style metal

3 · Clap — staggered band-passed bursts plus a tail

Method: noise shaping with structure. Three or four short bursts fired ~12 ms apart imitate many hands; a band-pass gives the "hand" colour; a short reverb glues the bursts into one clap.

white noise → band-pass
~1–1.5 kHz → 3–4 bursts
~12 ms apart → reverb tail
~200 ms → clap
Clap — Operator noise + band-pass + reverb
SourceWhite noise
FilterBand-pass, centre ≈ 1–1.5 kHz, moderate resonance
Burst envelopeA 0 ms · D 15 ms · S 0 (per burst)
Stagger3–4 MIDI notes 10–20 ms apart in the clip, last one loudest/longest
TailShort reverb, decay 150–300 ms, small room, on a short send
Mono checkConfirm the tail survives in mono — the clap reads as one event, not separate ticks

4 · Bass — FM at an integer ratio

Method: harmonic FM. A 1:1 or 1:2 ratio keeps it singable; the modulation index sets the brightness; an index envelope gives the attack bite without detuning the pitch.

modulator B
ratio 1 or 2 → modulates → carrier A
ratio 1 ↑ index env B amp env
fast decay → harmonic bass
Bass — Operator (1 : 1 or 1 : 2)
AlgorithmB → A (two-operator stack), C/D off
Carrier ASine, Coarse 1, level full
Modulator BSine, Coarse 1 (or 2 for hollow/odd emphasis), level = chosen index
Ratio1 : 1 (round) or 1 : 2 (hollow) — both harmonic, both singable
Index envelope (B amp)A 0 ms · D 120 ms · S low — bright attack, mellow body
Amp env (A)A 0 ms · D/S to taste · short R so notes leave gaps
GainCompensate A's output as you raise index, so brightness ≠ loudness

5 · Lead — FM, harmonic body with an inharmonic attack

Method: FM with a strong index envelope. Integer ratio keeps the body pitched; a brief high index at the attack adds a metallic "tine" that fades — the classic DX e-piano structure.

modulator B
ratio 1 → modulates → carrier A
ratio 1 ↑ steep index env B env
150 ms decay → e-piano-like lead
Lead — Operator (1 : 1, enveloped index)
AlgorithmB → A, C/D off (add feedback on B for edge)
Ratio1 : 1 harmonic; push B to ≈ 1.41 for a bell/mallet variant
Index env (B amp)A 0 ms · D 150 ms · S low — the "tine fades to body"
Amp env (A)A 0 ms · sustained, medium release for a struck-but-singing feel
Feedback (B)seasoning only — small amounts add edge, large amounts push toward noise
Watch forAliasing on high notes when index is high — lower index or play lower

6 · Pad — additive-style wavetable morph

Method: additive-style. A slow sweep through the wavetable position redistributes the partials, so the timbre evolves with no new notes — movement from the spectrum, not the arrangement.

wavetable osc → slow LFO
→ table position → gentle LP filter → long amp env → evolving pad
Pad — Wavetable
Oscillator 1A harmonically rich table (e.g. a "Basics" saw-to-sine sweep)
Table position modSlow LFO (≈ one cycle per 10–20 s), depth small — partials redistribute
FilterLow-pass, gentle, cutoff mid; let the table do the work, not the filter
Amp envA 300–800 ms · long S · long R — it swells, never clicks
VoicesHold two notes (root + fifth) for a stable bed
WidthStereo via Unison/Utility — mono-check it survives

7 · Drone — noise shaping or west-coast fold

Method A: filtered noise stretched into cloud. Method B (recommended for the texture): a folded sine through a low-pass gate, where brightness blooms with level. Pick one — don't stack.

VCO sine/tri → wavefolder
↑ env on fold depth → low-pass gate
vactrol, "both" ↑ slow ping → drone
Drone A — Wavetable noise
SourceWavetable noise oscillator
FilterBand-pass ≈ 1.2 kHz, moved slowly by a slow LFO
Amp envVery long A and R — a continuous bed
StretchIf printed, Warp mode Texture, half-speed for clouds
Drone B — VCV west-coast (recommended)
OscillatorVCO sine or triangle (harmonically poor on purpose)
WavefolderBogaudio FOLD / Befaco — fold depth from an attenuated AD envelope
Low-pass gateBogaudio LPG or Veils in LPG ("both") mode — vactrol curve
PingSlow trigger/clock; harder ping = brighter + louder + longer
The one moveDrive fold depth from a slow LFO/random — freeze it and the part should die
Decision check — before you commit a recipe
  • For each sound, can I draw its signal flow from memory — source → shaper(s) → output?
  • For each pitched sound, did the Spectrum confirm my harmonic/inharmonic prediction?
  • For the kick: can I point to where the click ends (ms) and the body begins?
  • For the drone: is exactly one control carrying the movement, or did I stack three?
