Synthesis from Zero · 04 — Setup

The synthesis rig

For synthesis from zero, the rig is mostly software — the sounds are generated, not recorded, so cables matter less than knowing which device does which job. This page lays out the instruments, the signal path inside the box, and the one analyzer that turns "harmonic vs inharmonic" from a feeling into a picture.

Ableton Live 12 Suite

  • The studio brain — hosts every synth, records, arranges
  • Operator (FM/PM), Wavetable (additive-style), Drum synths, Sampler
  • Spectrum analyzer — see the harmonics, don't only hear them

Operator

  • Four-operator phase-modulation FM synth with selectable algorithms
  • Carrier:modulator ratio + index = predictable spectra
  • Also a sine source with its own pitch envelope — the kick body lives here

Wavetable

  • Two wavetable oscillators + sub + noise, two filters
  • Sweeping the wavetable position = additive-style spectral morph
  • Built-in noise generator feeds the hat and clap recipes

Live drum synths

  • Drum Rack hosting Operator/Wavetable per pad — no sampled pads
  • Each voice is one synth instance: kick, hat, clap
  • Routed to a SYNTH KIT group for one drum bus

Arturia MicroFreak

  • Hardware oscillator engines + analog-style filter + wavefolder
  • The analog alternative path for bass, drone, and folded tones
  • One envelope only — note its limit for separate-click drum work

VCV Rack 2 (optional)

  • Free modular — the only practical home for a true low-pass gate
  • VCO → wavefolder → LPG for the west-coast voice
  • Recorded into Live over the audio interface, or via its VST in Suite

The signal path for synthesis work

Two paths run in parallel — generated audio inside the box, and an analyzer on the master so you can verify the spectrum. The MicroFreak and VCV only enter when you want the analog folder or the true LPG; everything else is internal.

Internal synthesis path (the default)

Operator / Wavetable → per-track FX → group bus → Master + Spectrum → interface → DT 990 Pro

External path (only for the folder / LPG voice)

VCV: VCO → folder → LPG → audio out → interface in → Ableton audio track + MicroFreak Out → interface In 1
The one device you must keep open

Put Live's Spectrum on the master and leave it there for the entire build. This Lab Note's promise — predict the spectrum from the patch — is only verifiable if you can see harmonic (evenly spaced spikes) versus inharmonic (uneven spikes) versus noise (a continuous band). Trust your eyes to confirm what your ears predict.

Gain & metering plan

Synthesis has one gain trap that recording does not: raising an FM index or a fold depth also raises level, so a "brighter" change can be just a "louder" change in disguise. Gain-compensate every spectral move so you judge timbre, not volume.

StageTargetIts one job
Each voice, solopeaks ≈ -12 dBFSLeave headroom so the group sum doesn't clip. Set the synth output, not the fader.
Index / fold changesloudness-matchedWhen raising FM index or fold depth, trim the operator/output level so only brightness changes — A/B at matched volume.
Group bus (SYNTH KIT, SYNTHS)peaks ≤ -6 dBFSOne place to balance the kit against the tonal parts.
Masterpeaks ≤ -3 dBFSNo limiter for this exercise. The headroom is part of the sound.
Phones knobcomfortable, then leave itListening level only — zero effect on disk. Riding it while patching destroys your judgement.
The golden rule of synthesis gain

A spectral parameter that also changes loudness is lying to you. Compensate at the source so "brighter" never secretly means "louder" — otherwise the louder version always wins, and you learn nothing about the spectrum.

Latency, clock, and buffer

Buffer size at 48 kHz — pick per task
128 samplesUse when playing a synth live and the latency feels sluggish.
256 samplesThe default. Most of this work is programmed, so latency barely matters.
512+ samplesMix and export sessions, and when running VCV alongside Live — CPU headroom over latency.
  • The MicroFreak's one envelope is its real limit for drum synthesis: you cannot easily run a separate pitch envelope and a separate click envelope at once. For the kick's two-generator structure, prefer Operator; use the MicroFreak as the analog-flavour alternative noted in each recipe.
  • VCV is optional but recommended for the west-coast voice — it is the only easy place to get a true vactrol-modelled low-pass gate. If you skip it, the MicroFreak's wavefolder plus a tight envelope is the documented approximation.
  • Synthesized one-shots may be printed as WAVs. That is not "using samples" — it is bouncing your own synthesis. The constraint is no borrowed recordings, not no audio files.

Tempo & mood

Recommended tempo: 120–126 BPM. Start at 122 BPM.
