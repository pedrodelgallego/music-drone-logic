03 — Studio & setup

Your studio

Five boxes and one pair of headphones. Every step is written against exactly this setup — and this page explains every cable, every gain stage, and every clock signal between them.

Ableton Live 12 Suite

  • The studio brain — records, edits, arranges
  • Audio takes, effects, automation, mixing
  • Master MIDI clock for both hardware boxes

Elektron Digitakt II

  • 16-track stereo drum computer and sampler
  • Kick, hats, percussion, resampled texture
  • Parameter locks, trig conditions, retrigs, mutes

Arturia MicroFreak

  • Digital oscillator engines + analogue-style filter
  • Source of bass, drone, and texture layers
  • Onboard sequencer and modulation matrix

Launchpad Pro MK3

  • 64 RGB pads + internal step sequencer
  • Ableton clip launching + track mutes
  • Live automation gestures via MIDI mapping

MOTU M4

  • 4-in / 4-out USB-C audio interface
  • Inputs 1/2: front combo preamps with gain knobs
  • Inputs 3/4: rear line inputs, full-screen metering

Beyerdynamic DT 990 Pro

  • Open-back, 250 Ω studio headphones
  • Wide soundstage, honest about depth
  • Watch for the upper-mid bump near 7–10 kHz

How the boxes talk to each other

Two flows run in parallel — audio and control. Knowing which signal lives where prevents most beginner debugging trips. Read this section once before touching a cable; refer back whenever something is silent, doubled, or out of time.

Audio path

MicroFreak Out → TS → M4 In 1 + Digitakt II Main L/R → 2 × TRS → M4 In 3/4 → USB-C → Ableton → USB-C → M4 Phones → DT 990 Pro
Recommended path

Use the MOTU M4 as the single audio device in Ableton: MicroFreak mono into input 1, Digitakt II stereo into inputs 3/4, headphones from the M4. One audio device, one clock, no aggregate-device complexity.

  • TS vs TRS: the MicroFreak output is unbalanced, so a plain TS guitar-style cable is correct there. The Digitakt II main outs are impedance-balanced — TRS cables on those two runs reject hum over longer distances. TS cables also work; you just lose the noise rejection.
  • Why input 1 and not 1/2: the MicroFreak is mono. Recording it to a mono Ableton input keeps the file half the size and forces you to create width deliberately (with effects) instead of accidentally.
  • Keep audio and power cables apart. If you hear a faint whine, the usual suspect is an audio cable lying across a power supply or a USB hub.

The gain chain — where level is set, in order

Audio passes through five level controls before it reaches your ears. Each one has exactly one job. Set them in this order, left to right, and never fix a problem at the wrong stage.

StageControlSet it toIts one job
1 — SourceMicroFreak Master Volume~ 70–80 %Send a healthy line-level signal. Too low and you amplify noise later; full clockwise can sound brittle into some inputs.
1 — SourceDigitakt II Main Volume~ 12 o'clock, then adjustInputs 3/4 on the M4 have no gain knobs — they are fixed line inputs. The Digitakt's own volume is the record level for the drums. It can also out-shout them: Elektron outs peak hotter than the 3/4 inputs accept, so near-full main volume will clip the converter with no knob to save you.
2 — PreampM4 Input 1 gainMeter green, occasional amber, never redMatch the MicroFreak to the converter. Watch the M4's on-screen meters while holding a loud note.
3 — Converter → DAWAbleton input meterPeaks -12 to -6 dBFSConfirm what is actually being written to disk. This is the number that matters; everything before it serves this.
4 — MixAbleton track fadersSet in the mix pass (Step 09)Balance recorded material. Faders never change what was recorded — only what you hear.
5 — EarsM4 Monitor / Phones knobsComfortable, then leave itListening level only. It has zero effect on the recording. Riding this knob while mixing destroys your level judgement.
The golden rule of gain

Fix level problems at the earliest stage that causes them. If Ableton's meter is low, turn up the synth or the preamp — never the track fader. If your ears hurt, turn down the phones knob — never the synth.

Monitoring & latency

SituationMonitor throughWhy
Designing sounds, jamming, sequencing hardwareM4 direct monitoring (the per-channel MON buttons) with the Ableton track Monitor set to OffZero latency — the input is routed to your headphones inside the interface, before the computer. Knob moves feel instant.
Recording a takeAbleton track Monitor on Auto, M4 MON off for that channelYou hear exactly what is being recorded, including any Ableton effects. Never both at once — both at once = a flanged double.

Buffer size at 48 kHz — pick per task
128 samples~ 2.7 ms buffer each way. Use when software-monitoring through Ableton effects feels sluggish.
256 samples~ 5.3 ms each way, ~ 12–15 ms round trip. The course default — stable and low enough for everything here.
512+ samplesUse for the mix and export sessions. Latency no longer matters; CPU headroom does.

Audio connection options

OptionUse it when
Option A — MOTU analog, recommendedMicroFreak Audio Out → M4 Input 1. Digitakt II Main L/R → M4 Inputs 3/4. Ableton input/output device: MOTU M4. This is the simplest, most stable setup for the course.
Option B — Digitakt II USB audioDigitakt II connects by USB and sends audio directly to the computer. Use this if you want one cable for Digitakt audio/MIDI, but avoid recording the same Digitakt signal from both USB and M4 at once.
Option C — Overbridge laterUse Overbridge when you want deeper computer integration or separate Digitakt tracks in Ableton. Keep the first build on the analog route so routing does not become the lesson.
Option D — MicroFreak into Digitakt IIMicroFreak Audio Out → Digitakt II Inputs. Use this when you want to sample the MicroFreak into Digitakt II, resample it, sequence it, or process it before Ableton.
Option E — macOS aggregate deviceUse Audio MIDI Setup only if you need M4 and Digitakt II USB audio available at the same time. This adds complexity and is not needed for the first track.
Avoid doubled audio
  • If Digitakt II is connected by both audio cables and USB, decide which path Ableton is recording.
  • Doubled monitoring causes comb filtering, hollow drums, or a tiny slap delay.
  • For this course, record Digitakt II from M4 Inputs 3/4 and use USB for MIDI clock unless you deliberately switch to USB audio.
  • Same trap on the synth: if a channel's M4 MON button is lit and the Ableton track Monitor is In/Auto with the track armed, you hear it twice. One or the other.

One clock to rule them all

Exactly one device generates tempo. Everything else listens. If the Digitakt's screen shows a tempo different from Ableton's, clock sync is broken — stop and fix it before recording anything, because a take recorded against a drifting clock cannot be repaired.

  • USB clock has small jitter (well under a millisecond on a direct port). For this music it is inaudible. Plug both hardware boxes directly into the computer, not through an unpowered hub, to keep it that way.
  • Hardware starts can land a few milliseconds late relative to bar 1 in Ableton. Don't fight it during recording — record long takes, then nudge the clip to the grid afterwards. Step 02 of the build shows how to measure and compensate.
