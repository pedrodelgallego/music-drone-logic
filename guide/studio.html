03 — Studio & setup

Your studio

Four boxes and one pair of headphones. Every step is written against exactly this setup.

Ableton Live 12 Suite

  • The studio brain — DAW, hosts everything
  • Drum Racks, audio takes, effects, automation
  • Arrangement, mixing, exporting

Arturia MicroFreak

  • Digital oscillator engines + analogue-style filter
  • Source of bass, drone, and texture layers
  • Onboard sequencer and modulation matrix

Launchpad Pro MK3

  • 64 RGB pads + internal step sequencer
  • Tactile drum programming + performance mutes
  • Live automation gestures via MIDI mapping

MOTU M4

  • 4-in / 4-out USB audio interface
  • Routes MicroFreak in, monitors out
  • Zero-latency direct monitoring for tracking

Beyerdynamic DT 990 Pro

  • Open-back, 250 Ω studio headphones
  • Wide soundstage, honest about depth
  • Watch for the upper-mid bump near 7–10 kHz

How the boxes talk to each other

Two flows run in parallel — audio and control. Knowing which signal lives where prevents most beginner debugging trips.

Audio path

MicroFreak → MOTU M4 In 1 → Ableton → MOTU Out → DT 990 Pro

Control + MIDI clock

Launchpad Pro MK3 ↔ USB ↔ Ableton → MIDI clock → MicroFreak
