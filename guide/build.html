06 — The build

Step-by-step

Only now do we touch the gear. Each step is small. Each step has a decision checkpoint.

Step 01

Create the Ableton project

Open a new set
  • ⌘ N for a new Live Set. Press Tab to switch to Arrangement view.
  • Set tempo to 124, time signature 4/4.
  • Right-click the timeline ruler → Time Ruler Format → Time to see positions as m:ss.
Audio preferences
  • ⌘ , → Audio. Input + Output device: MOTU M4.
  • Sample Rate 48000 Hz, Buffer Size 256 samples.
  • Enable Input Config channels 1/2 mono and stereo.
Enable the Launchpad Pro MK3
  • Plug the Launchpad in via USB. Open ⌘ , → Link, Tempo & MIDI.
  • Control Surface: Launchpad Pro MK3. Input + Output: LPProMK3 DAW.
  • Enable Track and Remote for the LPProMK3 MIDI input — note data from Note and Sequencer modes.
  • Press Session on the Launchpad — pads light up confirming the link.
Send MIDI clock to the MicroFreak
  • Same Preferences tab. Find the MicroFreak port → enable Sync on the output row.
  • MicroFreak: Shift + Tempo → source USB. The synth now follows Ableton.
Create the tracks
  • ⌘ ⇧ T three times for MIDI tracks: KICK, HAT, PERC.
  • ⌘ T three times for audio tracks: MF BASS, MF DRONE, ATMOS.
  • ⌘ ⌥ T twice for return tracks: A — REVERB, B — ECHO.
  • Rename with ⌘ R. Color-code tracks. ⌘ S as hypnotic_lab_001.als.
Step 02

Connect and gain-stage the MicroFreak

Physical signal chain
  • 1/4" TS cable from MicroFreak Audio Out → MOTU M4 Input 1.
  • On the M4 front panel: hold MON, set Input 1 to Line.
  • Headphone output on the M4 → DT 990 Pro. Knob at halfway to start.
Configure the MF BASS track
  • Select MF BASS. ⌘ ⌥ I to show I/O.
  • Audio From → Ext. In, channel 1 (mono).
  • Monitor → In while dialing sounds. Switch to Auto before recording.
  • Arm the track (red circle, or F when selected).
Gain staging
  • Hold a note. Adjust the MicroFreak master volume until peaks land between -12 dBFS and -6 dBFS.
  • If you'd have to push the M4 input gain into the red, push more level out of the synth instead.
  • Test record: ⌘ ⇧ R, 8 bars. Play back. Confirm clean, no clipping, no buzz.
Step 03

Build the bass sequence

Choose the oscillator
  • MicroFreak Type encoder → Wavetable. Strong alternatives: Harmo, SuperWave, KarplusStr.
  • Shift + press Type to initialise.

Program the sequence
  • Length 16 steps. Scale: A natural minor. Root A1.
  • 3–5 notes only. Place the root on step 1, leave silences. Slide one or two steps for acid glides.
  • Sync MicroFreak to Ableton: Shift + Tempo → USB.
Record the take
  • Loop and listen for at least 3 minutes before deciding it works.
  • Arm MF BASS. Position playhead at bar 1. ⌘ ⇧ R. Record 4 minutes.
Decision check
  • Can I listen to this pattern for several minutes without skipping?
  • Does it create tension instead of resolving?
  • Is any step "decorative"?
  • Does it leave headroom for kick and drone?

Build the drone

Save bass, init drone
  • Shift + Memory → save bass as LAB BASS.
  • Init a fresh patch for the drone. Do not edit the bass patch.

  • Is the 80 – 200 Hz region muddy?

Sidechain the bass to the kick
  • Compressor on MF BASS. Sidechain on, Audio From KICK Pre FX.
  • Ratio 4:1, Attack 1 ms, Release 120 ms, ~ -3 dB reduction per kick.
Decision check
  • Does the groove move without thinking?
  • Is the kick burying the bass?
  • Is the hat supporting or fronting?
Step 06

Create the atmosphere layer

Pick one method. Don't stack them — one textured layer, not a wall.

Option A — Reversed drone
  • Select drone clip, ⌘ D, drag copy to ATMOS.
  • Right-click → Reverse.
  • Auto Filter HP @ 300 Hz, resonance 15 %. Reverb send -6 dB.
Option B — Resample reverb tail
  • Hybrid Reverb on master, decay 15 s, Dry/Wet 100 %.
  • New audio track, Audio From → Resampling. Arm. ⌘ ⇧ R. 16 bars.
Option C — Stretched noise
  • MicroFreak Oscillator Noise. Wave 80, Timbre 30. Record 60 s.
  • Clip view → Warp Mode Texture. Set clip BPM to 62.
  • Auto Filter BP @ 1.2 kHz → Reverb send -6 dB.
Option D — FX-only
  • Short drone fragment → Grain Delay: Freq 8.0, Pitch -7 st, Spray 60 ms, Dry/Wet 60 %.
  • Hybrid Reverb large algorithm, decay 10 s. EQ to live in the mids only.
Decision check
  • Muting the atmosphere should make the track feel smaller, not missing.
  • If you can clearly name what it's doing, it's too loud.

Perform automation on the Launchpad
  • ⌘ M for MIDI map. Click Auto Filter Cutoff → tap a Launchpad pad. Set Min/Max to 600 Hz / 3500 Hz.
  • Map four pads: cutoff, resonance, echo feedback, reverb send.
  • Exit map mode. Arm parameters for automation. F9 for Arrangement Record. Tap pads slowly.
  • After: right-click lane → Simplify Envelope.
Decision check
  • If you can hear the automation move, it's too fast.
  • Can you remove one lane without weakening the arc? If yes, remove it.

Clip placement, section by section
  • 0:00 – 1:00 — Drone from bar 1, atmosphere at -22 dB.
  • 1:00 – 2:00 — Kick MIDI from bar 32. First hit at velocity 70 (soft entrance).
  • 2:00 – 3:00 — Bass audio from bar 63. ⌘ E to split first 4 bars, fade in via clip corner.
  • 3:00 – 4:30 — Hat from bar 94, perc from bar 110. Bass filter opens 600 Hz → 3.5 kHz.
  • 4:30 – 5:30 — Nothing new. Energy comes entirely from automation peaks.
  • 5:30 – 6:30 — Delete perc, mute hat 32 bars, kick fader -2 dB via automation.
  • 6:30 – 8:00 — Cut bass, then kick at 7:00. Drone + atmosphere fade-out on master volume.
Rehearse the arc on the Launchpad first
  • Drop clips into Session view. Tab to switch views.
  • Session on Launchpad — 8×8 grid mirrors Session.
  • Use track-select row to mute/unmute live across the 8-minute arc.
  • When gestures feel right, replicate as clip placements with fades in Arrangement.
Polyrhythmic perc on Launchpad's internal sequencer
  • Select PERC. On Launchpad: Sequencer, Length 14 steps.
  • Place 2–3 hits. 14 against 16 phase-shifts slowly — drift without new notes.
  • Record into a long MIDI clip, trim where the drift is welcome.

Step 09

Basic mix pass

Group and reset
  • Select KICK, HAT, PERC → ⌘ G → DRUMS.
  • MF BASS, MF DRONE, ATMOS → ⌘ G → SYNTHS.
  • Pull all faders to -inf. Bring up one at a time, 30 s between each.
Target fader levels
Starting balance
Kick-10 dB (peaks ~ -6 dBFS)
Bass-12 dB
Drone-16 dB
Closed Hat-18 dB
Percussion-22 dB
Atmosphere-20 dB
Return A (Reverb)-8 dB
Return B (Echo)-10 dB
Master peak≤ -3 dBFS
Frequency separation
  • MF DRONE: EQ HP 120 Hz, 24 dB/oct.
  • ATMOS: HP 220 Hz, LP shelf 8 kHz -3 dB.
  • MF BASS: narrow bell cut 200 – 300 Hz by 1.5 dB if woolly.
Master bus check
  • Spectrum + Utility on master. Nothing else. No limiter for this exercise.
  • Flip Utility Width to 0 % every few minutes — anything disappearing in mono = widening overdone.
  • Take headphones off every 15 minutes. Ear fatigue is the main cause of bad mixes here.
