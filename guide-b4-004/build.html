Lab Note B4-004
Tape & Degradation · 06 — The build

Build a disintegration-loop track, step by step

Only now do we touch the gear. We will write a beautiful short loop, build a degradation pass so gentle it is almost inaudible, then print generation after generation against a written log, and finally arrange the decay itself as the form of the piece. Every step is small, every step has a decision checkpoint, and every value is spelled out. Keep the setup page open in another tab — it is the rig reference this page assumes.

Step 01

Set up the project and the parking lot

New set
  • ⌘ N for a new Live Set. Tab to Arrangement view — the whole piece lives here, not in Session.
  • Tempo 120 for now; the loop will not be beat-locked, but a tempo gives the grid for clean loop lengths. Time signature 4/4.
  • Right-click the ruler → Time Ruler Format → Time so positions read as m:ss. This is a long, slow piece; minutes matter more than bars.
  • ⌘ , → Record, Warp & Launch: File Type WAV, Bit Depth 24. Turn OFF Auto-Warp Long Samples — warping would smear the very pitch drift you are cultivating.
Create the tracks
  • ⌘ T for an audio track named LOOP — the live source, holding whichever generation is currently being degraded.
  • ⌘ T for PRINT — Audio From → Resampling, Monitor Off. This captures each new generation.
  • ⌘ T for GENERATIONS — a parking track where every printed clip is filed and never overwritten.
  • ⌘ ⌥ T for one return track A — SPACE with a Hybrid Reverb, decay 6–8 s, Dry/Wet 100 %. The decay needs a room to dissolve into.
Project hygiene
  • ⌘ S → save as labnote_b4_004.als in a dedicated project folder.
  • Create a folder gen_00_archive outside the live set. Generation 0 will be copied here and never reopened in the degradation chain — it is the only untouched original.
  • End every session with File → Collect All and Save.
Step 02

Write generation 0 — the thing worth losing

Choose the source
  • The loop must be short, pretty, and fully original — a drone chord, a melodic fragment, a sung phrase, an arpeggio. 4 to 8 bars. This is your "tape".
  • Recommended for a first build: a slow two-note pad on the MicroFreak (root + fifth, long attack and release) — it exposes pitch wobble and dulls gracefully. Record 8 bars into LOOP.
  • Whatever the source, it should be something you would happily loop untouched for two minutes. If it bores you pristine, it will be unbearable degraded.
Note the starting state precisely
Generation 0 reference — write these down
Peak levelSet the take to peak around -6 dBFS — headroom for every later saturation stage
Top endFull and present; note that gen 0 reaches the full spectrum. This is the ceiling everything erodes from.
Stereo widthNote whether it is mono or stereo; width only ever decreases from here
Loop lengthExact bars/seconds. Every print must be this same length or the generations will not stack cleanly.
Make the loop seamless and archive it
  • Trim the clip to exactly the loop length on bar lines. Add a 10–20 ms crossfade across the loop seam (drag the clip fade handles) so it cycles without a click.
  • Consolidate with ⌘ J into one clean clip. Drag a copy to GENERATIONS and label it gen 0.
  • Export this clip to gen_00_archive as gen_00.wav. From now on, the source in LOOP is allowed to die — this archived copy is not.
Decision check
  • Would I happily listen to this loop, untouched, for two minutes? If not, rewrite it before degrading — decay cannot rescue a dull source.
  • Does it loop seamlessly with no click at the seam?
  • Is gen 0 archived outside the set, so the contrast survives even when no copy does?
Step 03

Build the degradation pass

Drop the chain on LOOP — in order
  • On the LOOP track, build the chain from page 04: Saturator → Vinyl Distortion → Redux → Auto Filter (LP) → Echo → Utility. Order matters — saturate first, filter the result, then wobble and narrow.
  • This single chain is your tape head. It will be applied once per print, and because you reload each print as the new source, its effect compounds.
Dial each device for a barely-there single pass
Starting values — one pass should be almost inaudible
SaturatorSoft Sine, Drive +3 dB, Output -3 dB so the pass does not get louder
Vinyl DistortionCrackle Density 15, Crackle Volume low; Tracing/Pinch subtle
ReduxSample rate full at first — leave this idle until late generations
Auto FilterLP, 12 dB, cutoff just below the loop's top so it shaves only the highest highs (start ~ 14 kHz)
EchoWobble Amount 10 %, slow morph (wow); Noise Amount 8 %; Dry/Wet small so it does not add obvious delay
UtilityWidth 100 % now; you will lower it per pass
  • A/B the chain against bypass on a single play. If you can clearly say "that is degraded", you have gone too far for one pass — pull every device back until it is just a warmth.
Decision check
  • With the chain on, does one pass sound only slightly warmer and softer than gen 0 — not destroyed?
  • Is the lead symptom I chose on page 03 the one that is most present, with the others supporting it?
  • Is the output level matched to the input, so the decay's slope comes from damage, not from a level change I did not intend?
Step 04

Print the generations against a log

The print loop — mechanical, repeatable
  • Solo LOOP. Confirm only LOOP reaches the master; PRINT is on Resampling, Monitor Off.
  • Arm PRINT. Position the playhead at bar 1. ⌘ ⇧ R and record exactly one loop length. Stop. This is generation 1.
  • Drag the new print from PRINT to GENERATIONS, label it gen 1, then drag a copy onto LOOP, replacing the previous source clip. The chain now degrades the degraded copy.
  • Advance one symptom value per the escalation rules below, then print again for generation 2. Repeat.
Escalate gently — what changes between prints
Per-print adjustments on the LOOP chain
Every passAuto Filter cutoff down ~ 600 Hz; Utility Width down 8 %
Every passEcho Noise up ~ 3 %; Vinyl Crackle Volume up a hair
From gen 4Echo Wobble up to 18–25 % so pitch drift becomes audible
From gen 6Begin lowering Redux sample rate for grit; allow brief volume dips (dropouts)
Hold steadySaturator Drive — keep constant so the decay slope stays smooth (or ramp it for a lurching decay)
The generation log — keep it as you go
labnote_b4_004_genlog.txt — write a line per print
gen 0pristine · peak -6 dBFS · full top · stereo · ARCHIVED
gen 1LP 13.4k · width 92% · noise 8% · warmth only, still "clean"
gen 2LP 12.8k · width 84% · noise 11% · clearly "tape" now
gen 3LP 12.2k · width 76% · noise 14% · top dull, hiss rising, first wobble
gen 4LP 11.6k · width 68% · noise 17% · wobble 18% · pitch visibly drifting
gen 6LP 10.4k · width 52% · noise 23% · WEATHERED SWEET SPOT — mark for arrangement
gen 8LP 9.2k · width 36% · noise level w/ signal · Redux engaged · dropouts begin
gen 10LP 8k · near mono · melody dissolving · Basinski zone
gen 12more gap than tone · an impression of the loop · DUST
  • The log is not bookkeeping — it is the map of the piece. When you arrange, you will pull specific generations from GENERATIONS by their logged character, not by ear-hunting through identical-looking clips.
  • Print at least to your target generation plus two, so you have material past the endpoint to choose from.
Decision check
  • Does each generation sound like a survivor of all the ones before it — does gen 6 sound like it lived through 1–5, not just "a filtered loop"?
  • Is the loss genuinely compounding? Spot-check: gen 6 should be dramatically further from gen 0 than gen 3 is. If gens 3 and 6 sound similar, you are re-degrading the original by accident — confirm you reload the print.
  • Is there any way back? There should not be. The only escape hatch is the archived gen 0.
  • Have I logged every pass, so I can find the weathered sweet spot and the dust by name later?
Step 05

Optional — a real-tape generation

Run one pass through the cassette deck
  • If you have the Path A rig from page 04, route a mid-generation print (say gen 4) out to the cassette deck and back, as wired on the setup page.
  • Record the return into PRINT and file it in GENERATIONS as gen 4-tape. Note in the log that this pass added flutter and oxide hiss no plugin gave you.
  • Continue the in-the-box chain from that tape print if you want the genuine flutter baked into everything downstream.
Decision check
  • Does the real-tape pass add irregular flutter and character the emulation could not? If it sounds the same as a plugin pass, the deck is not earning its place — skip it.
  • Did the deck destroy the loop in one pass? Lower its record level and clean the heads, then retry from the same source generation.
Step 06

Arrange the decay as the form

Decide the shape of the piece
  • The composition is the sequence of generations over time. The simplest and strongest form is monotonic: start near gen 0, end at your target generation, and let the loop simply repeat and worsen across 6–9 minutes.
  • You are not adding sections or new elements. The only event is the slow handover from one generation to the next. This is the thesis of the whole Lab Note in one arrangement.
Place the generations on the timeline
A monotonic disintegration arc — time → generation
0:00 – 1:30gen 0 / gen 1 — the loop pristine, the listener settling in, learning the phrase
1:30 – 3:00gen 2 / gen 3 — warmth and hiss creep in; "is this getting older?"
3:00 – 4:30gen 4 / gen 6 — clearly worn; the weathered sweet spot; pitch wavering
4:30 – 6:00gen 8 — melody dissolving into texture; reverb send opening to catch the dust
6:00 – 7:30gen 10 / gen 12 — more gap than tone; the loop becomes an impression
7:30 – 8:00silence, or gen 0 returning for a single bar — your choice of ending
Crossfade between generations, never cut
  • Lay each generation's clip on LOOP (or alternating lanes) so consecutive generations overlap by one loop length. Drag the fade handles to make a long crossfade across the seam.
  • A hard cut between gen 3 and gen 4 announces "something changed" — exactly the event you are trying to hide. The decay should be felt, never witnessed.
  • Loop a suspicious crossfade with ⌘ L and listen three times. If you can name the moment the generation changed, lengthen the fade.
One slow gesture of space
  • A to show automation. Draw the A — SPACE reverb send rising slowly from -12 dB at 0:00 to -3 dB by 6:00, so the room blooms exactly as the loop loses its body. The reverb catches the dust as the signal lets go of it.
  • No ramp shorter than 16 bars. If you can hear the send move as an event, it is too fast.
  • Resist adding anything else. One reverb gesture is the entire "production" — the decay does the rest.
Decision check
  • Play 0:00 then jump to 6:00 — is the distance shocking? It should be. The whole piece is that gap, revealed slowly.
  • Can I hear any individual generation handover as a cut? If yes, lengthen the crossfade until it disappears.
  • Does the piece still hold without any new element entering? If it sags, the problem is the source loop or the decay slope — not a missing part. Resist the urge to add.
  • Is my chosen ending earned — silence after dust, or gen 0 returning as memory? Either works; pick on purpose.
Step 07

Mix check, export, and archive

Light mix pass
  • This is barely a mix — the decay is the mix. Put an EQ Eight on LOOP with a gentle HP around 30 Hz to keep rumble out, and nothing else unless something is genuinely wrong.
  • Watch the master meter across the whole piece: peaks ≤ -3 dBFS, including the reverb bloom. The rising hiss should never push the master into the red.
  • Mono check: flip a Utility on the master to mono every few minutes. Because the generations march toward mono anyway, the late piece should survive perfectly; only the early generations have width to lose.
  • Listen once quietly — at low volume the balance of signal-to-hiss across the arc shows honestly.

Archive the whole chain of evidence
  • File → Collect All and Save — every generation print now lives in the project folder.
  • Keep the generation log .txt beside the WAV. The track now exists as three things: the archived pristine gen 0, the full ladder of generations, and the final arranged render.
  • Copy the render to your phone. Listen in another room. Note in 10 lines or fewer where the decay sags or rushes — timestamped, not feelings. Wait a day, change one thing (usually the decay slope or one crossfade), render v2, compare.
  • What you just made: a piece whose only event is the erosion of a single loop, where the path's history is the entire composition. Nothing was added; something was lost, slowly, on purpose, and the listener felt every generation of it.
