Lab Note B4-004
Tape & Degradation · 04 — Setup

Build the degradation rig

There are two genuine ways to walk this path: a real cassette 4-track that physically loses material on every pass, or an in-the-box resampling chain that emulates the loss and lets you log every step. This page wires up both, and sets the gain discipline that keeps the decay reading as smooth rather than as a series of accidents.

The signal path, abstractly

source loop → degrade pass record → new copy reload → degrade pass …→ gen N
The whole rig in one sentence

Whatever the gear, you are building a loop that feeds its own output back into its input through something lossy, and printing a copy each time round. Everything below is just two ways to build that loop — and the discipline to make each lap small.

Path A — real cassette 4-track or dictaphone

Path A

The genuine article

What you need
  • A cassette 4-track (Tascam Portastudio or similar), or a handheld dictaphone, or any consumer tape deck with a record function.
  • A blank cassette — the cheaper and more worn, the more character. Type I (ferric) hisses more than chrome; lean into it.
  • Cables from your interface output to the deck's line/mic input, and from the deck's output back to an interface input.
Wiring — the round trip
  • Interface Output → 4-track Line In (or play the loop on a phone into the mic). This is how generation N enters the tape.
  • 4-track Line Out / headphone out → interface Input. This is how the freshly-degraded copy comes back into Ableton to become generation N+1.
  • Set the deck's record level so a loud part of the loop pushes the meter into the top quarter but not into the red. Hotter = more saturation per pass.
  • If your deck has Dolby noise reduction, switch it off — you want the hiss, not a circuit fighting it.
The physical loop method (Basinski-style)
  • For the genuine disintegration: record the loop onto a short length of tape, splice it into a literal loop, and let it run continuously past the head while you record the output. The oxide wears with every revolution.
  • Lower-effort version: record the loop to the cassette, play it back while recording to Ableton, then play that recording back into the deck again. Each bounce is one generation living on tape.
  • This path is slow, noisy, and unrepeatable — that is the point. Capture long passes; you cannot redo a take exactly.
Decision check
  • Does a single pass through the deck add audible hiss and a touch of wobble, but still leave the loop fully recognisable? If it is destroyed in one pass, lower the record level and clean the heads.
  • Can I hear flutter — fine pitch warble — that no plugin gave me? That irregularity is what justifies the cabling.

Path B — Ableton emulation chain

Path B

A resampling degradation chain

The copy mechanism
  • Create one audio track named LOOP holding generation 0, and a second audio track named PRINT.
  • On PRINT, set Audio From → Resampling. Each record pass on this track captures the master output — your "copy" mechanism. One pass = one generation.
  • Mute everything except LOOP while you print, so the resample captures only the degrading loop and nothing else.
The degradation device chain on LOOP
Stock Ableton devices, in this order
1 — SaturatorTape-style softening — the nonlinear medium. Soft Sine curve, Drive low (+2 to +4 dB). Adds harmonics and tames peaks.
2 — Vinyl DistortionCrackle and surface noise. Tracing Model + Pinch settings low; raise the Crackle Density and Volume a little for dust per pass.
3 — ReduxBit / sample-rate reduction. Drop sample rate gently for a grittier, dustier loss; keep bit depth high at first or it crushes too fast.
4 — Auto Filter (LP)Roll a few hundred Hz off the top each generation to mimic high-frequency loss. This is the "duller every copy" knob.
5 — Echo (wobble engine)Engage Echo's Wobble and Modulation for flutter, and its built-in Noise for hiss — wow/flutter and noise floor without a tape plugin.
6 — Utility (width)Narrow the stereo image a little each pass to simulate the centre-collapse of real generation loss.
  • If you own a dedicated tape emulation, it can stand in for steps 1, 4, and 5 — but the stock chain above is fully sufficient and more transparent to log.
  • Set every device so a single pass is barely noticeable. The drama comes from repetition, not from one extreme setting.
Print, reload, repeat
  • Solo LOOP. Arm PRINT (Resampling). ⌘ ⇧ R and record exactly one loop length. This is generation 1.
  • Drag the new print back onto LOOP as the source, replacing generation 0. The chain now degrades the already-degraded copy. Print again for generation 2.
  • Keep every print as a separate clip in a parking track — the sequence of generations is the composition. Never overwrite a print you might arrange later.
  • Archive generation 0 in a folder outside the set before you start. It is the only copy that survives untouched.
Decision check
  • Does one print sound only slightly worse than its source — not destroyed? If gen 1 is already harsh, pull every device back.
  • Does the loss compound when I reload and reprint, or am I accidentally re-applying the same gentle FX to the original each time (reversible)? Confirm you are reloading the print, not gen 0.

Gain discipline across passes

Record level is the decay's slope
Consistent level every passThe decay reads as a smooth, even slide. Set the print level once and never touch it.
Rising level across passesEach copy saturates harder than the last — the decay accelerates and lurches. Use deliberately for "the machine is breaking down".
Falling level across passesThe loop quietly recedes into its own hiss — a fading-memory feel that approaches silence rather than noise.
Peak targetKeep peaks around -6 dBFS on the print so each pass has headroom for the next stage's saturation. A pass that clips is a pass you redo.
Level is sound design again

Generation loss interacts with everything: each pass through a saturating stage at a hot level degrades faster and harder. The record level is not a housekeeping detail here — it is the single biggest control over the shape of the decay curve. Decide it on purpose.
