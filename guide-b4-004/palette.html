Tape & Degradation · 05 — Degradation palette

Recipes for wear

A reference of concrete settings for each symptom of decay, and how those settings should change pass over pass. Treat these as starting points, not law — but if a value drifts far from here, ask why. The goal is a single pass so subtle you can barely hear it, accumulating into something undeniable by generation six.

Symptom recipes — per single pass

Wow & flutter

  • Echo → Wobble Amount low; Wobble Morph toward slow for wow, fast for flutter
  • Or LFO → a pitch/warp parameter, sine, ~0.2–2 Hz, tiny depth
  • Aim for "is that drifting?" not "that is drifting"

Hiss

  • Echo → Noise Amount low, Noise Morph for colour
  • Or a quiet noise sample looped under the loop, level rising per gen
  • Hiss should sit just under the signal at gen 1, level with it by gen 8

Saturation

  • Saturator, Soft Sine, Drive +2 to +4 dB per pass
  • Output trimmed so the pass does not get louder
  • Rounds transients and thickens body without obvious distortion

Crackle & dropouts

  • Vinyl Distortion → Crackle Density low, Volume low
  • For holes: brief volume dips drawn in, or Redux glitches
  • Irregular and sparse — one or two events per loop, no more

The per-pass settings table

One generation — barely-there starting values (Path B)
Saturator Drive+3 dB, Soft Sine, output -3 dB to compensate
Auto Filter LP cutoffLower by ~400–600 Hz from the previous pass's setting
Echo WobbleAmount 8–15 %, slow morph (wow); raise morph for flutter
Echo NoiseAmount 5–10 %, rising ~3 % per generation
Vinyl Distortion CrackleDensity 10–20, Volume low; raise gently per generation
Redux sample rateFull at gen 0; nudge down only past gen 4 when you want grit
Utility Width-5 to -10 % per pass, marching toward mono
The rule behind the numbers

No single value here is loud. The craft is that each is set so one pass is almost inaudible, yet the symptoms all point the same direction — duller, narrower, noisier, wobblier — so they reinforce each other across generations. If any one value is obvious on a single pass, halve it.

How the settings escalate — the generation curve

What the loop sounds like at each milestone
Gen 0Pristine. Full top end, full width, no noise. The reference you archive and never touch.
Gen 1–2A warmth and a faint hiss. Most listeners would call it "analog", not "damaged".
Gen 3Top end noticeably duller, hiss rising, pitch beginning to waver. The character is now clearly tape.
Gen 6Recognisable but worn — the "weathered" sweet spot many tracks live in. Width has collapsed toward mono.
Gen 8–10Melody dissolving into texture, noise level with signal, dropouts appearing. The Basinski zone.
Gen 12+Approaching dust. More gap than tone; the loop is now an impression of itself.
Mark your destination

Cross-reference this against the endpoint you chose on page 03. If you want "weathered, still beautiful", you are arranging generations 1 through 6 and stopping. If you want full disintegration, you print all the way to 12+ and the late generations become the climax. The build page assumes you have a target generation in mind.

Lead-symptom palettes

Wobble-forward

  • Push Echo Wobble and pitch drift; keep filter and noise modest
  • Feels like a failing motor — seasick, queasy, unstable
  • Best on tonal, sustained material where pitch error is exposed

Noise-forward

  • Push hiss and crackle hard; keep pitch stable, filter gentle
  • Feels like an old, loved, much-played recording
  • The signal slowly drowns rather than warps — Belong, Basinski

Dullness-forward

  • Lead with the LP filter; drop several hundred Hz per pass
  • Feels like the recording sinking underwater or behind a wall
  • Quiet, mournful — the Caretaker's later stages

Dropout-forward

  • Lead with holes, stutters, and gaps; keep tone otherwise intact
  • Feels like a damaged disc or a tape with bald patches
  • Closest to glitch — Oval, Yasunao Tone; use sparingly
Before you build
  • I have a barely-audible per-pass setting for each symptom I am using.
  • I know which symptom leads, and I have pulled the others back so they support it.
  • I have a target generation from the curve above that matches my page-03 endpoint.
  • Every single value is subtle enough that one pass is almost inaudible.
