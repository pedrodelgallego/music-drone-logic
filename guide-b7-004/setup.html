Set Structure & Rehearsal · 04 — Setup

The performance rig, certified

A studio rig records; a performance rig has to survive a room, a clock, and your own adrenaline. This page lays out the live signal path, the single-clock rule, and the certification you run before any musical rehearsal — plus the physical mapping of the panic-path controls you will drill on page 06.

The performance rig

Ableton Live 12 Suite

  • The clock master and the arrangement spine
  • Holds the set map, return effects, the limiter
  • If it freezes, the hardware keeps its last state

Elektron Digitakt II

  • The groove engine — kick, hats, perc, texture
  • Mutes and pattern changes are your fastest live moves
  • Most likely to need a mid-set reboot; drill it

Arturia MicroFreak

  • Bass, drone, and texture voices
  • Synced LFOs do the slow work while your hands rest
  • A single mono cable — easy to lose, easy to mute cleanly

Launchpad Pro MK3

  • Mutes, scene launches, and the mapped ride controls
  • Holds the mute / kill / recover surface
  • Layout is muscle memory — never rearrange it before a show

MOTU M4

  • The single audio device in and out of Ableton
  • Feeds the house via balanced outs 1/2
  • Carries the resampling round-trip you certify nightly

Monitoring

  • DT 990 Pro for cueing; stage wedge or IEM for the room
  • Know which one tells the truth about low end tonight
  • Cue feedback before it reaches the house, not after

The live audio path

MicroFreak Out → TS → M4 In 1 + Digitakt II L/R → TRS → M4 In 3/4 → USB → Ableton (limiter) → USB → M4 Out 1/2 → House PA
One clock to rule them all

Exactly one device generates tempo — Ableton. The Digitakt II and MicroFreak receive clock and transport; nothing else sends. If the Digitakt's screen shows a tempo different from Ableton's, sync is broken — fix it before the doors open, never during the set.

The control + clock path

Launchpad Pro MK3 ↔ USB ↔ Ableton → clock + transport → Digitakt II + MicroFreak

The master limiter — the safety net before the room

  • A limiter sits last on the Ableton master, ceiling around -0.3 dBFS. It is not a mixing tool here — it is the wall between a runaway feedback path and the house system.
  • Set its ceiling once and freeze it. The limiter must catch the worst case you can produce, including the kill path failing to engage in time.
  • Every mapped ride control has a Min/Max range so the dangerous values are physically unreachable from the pad. The encoder cannot send the filter into a scream because you defined the scream out of its travel.

Rig certification — prove it before you play it

A rig diagram is a claim. Certification is checking reality against the claim, cable by cable, before any musical rehearsal.

Cert 01

Signal & clock pass

Prove every path in isolation
  • One master clock only. Press play in Ableton; confirm the Digitakt II and MicroFreak each start, stop, and read the set tempo. Stop transport; confirm both stop.
  • Trace audio end to end: MicroFreak alone to the PA, Digitakt L/R alone, the resample return alone. Hear each in isolation before hearing them together.
  • Confirm the resampling round-trip latency holds with tonight's cabling — a longer cable run or a different interface buffer changes it.
Cert 02

Gain staging & ceilings

Set the limits, then freeze them
  • Play the busiest moment of the set — full groove, peak filter, wettest send — and set gain so it peaks with headroom into the limiter, not at the rail.
  • Confirm the limiter ceiling and every encoder's Min/Max range. The danger values must be unreachable from the hardware.
  • Once set, gain staging is frozen. Re-checking it live is not a thing you do — riding the master during a set destroys your level judgement.
Cert 03

Panic-path mapping

Fix the three controls and find them blind
  • Mute: the per-track mute pads on the Launchpad. Memorise the position of each voice's pad so you can drop any element without looking.
  • Kill: one dedicated pad mapped to the feedback / resample channel mute, plus the master limiter as the backstop. This is the instant-silence path.
  • Recover: one pad or scene that lands the rig on the drone-and-kick floor. Map it, label it, and reach for it in the dark until it is reflex.
  • Close your eyes and find all three. If your hand hunts, move the controls until it does not.

Certification check
  • Is there exactly one clock master, confirmed starting and stopping both boxes?
  • Did I hear every audio path in isolation, then together?
  • Is the limiter ceiling frozen and every encoder range bounded away from danger?
  • Can I find mute, kill, and recover with my eyes shut?
