Lab Note B6-002
Self-playing Patches · 06 — The build

Build a self-playing patch and capture ten minutes of it

This is the deep page. We build one complete self-player from an empty project, prove it moves with your hands off, run the ten-minute unattended test, and capture the take. The worked rig is the MicroFreak matrix recorded into Ableton via the MOTU M4, with VCV Rack and Max for Live noted at each step. Keep the setup page open — it is the wiring reference this page assumes.

Step 01

Prepare the recording project

Open a clean set
  • ⌘ N for a new Live Set. Tab to Arrangement view — the ten-minute capture lives here, not in Session.
  • Tempo is irrelevant for a free-running self-player, but set 120 so the metronome and grid don't distract you. You will never sync anything to it.
  • Right-click the timeline ruler → Time Ruler Format → Time so positions read as m:ss. You are recording minutes, not bars.
Audio preferences
  • ⌘ , → Audio. Input and Output Device: MOTU M4.
  • Sample Rate 48000 Hz, Buffer Size 256 samples. Latency does not matter — nothing is monitored through Ableton in performance.
  • Input Config: enable mono input 1 (MicroFreak). Rename it MicroFreak.
  • ⌘ , → Record, Warp & Launch: File Type WAV, Bit Depth 24. Turn OFF Auto-Warp Long Samples — a self-player has no tempo to warp to and warping would smear the drift.
One audio track, armed and waiting
  • ⌘ T for one audio track, named HABITAT. Audio From → Ext. In, channel 1.
  • Monitor → In while you design, so you hear the MicroFreak. Switch to Auto before recording.
  • ⌘ S → save as self_player_b6_002.als in a dedicated folder. Takes record into its Samples/Recorded folder automatically.
In-the-box and VCV alternatives
  • VCV Rack: no Ableton needed yet — you will record with VCV's own recorder module, or route its output into this same HABITAT track via an aggregate device.
  • Max for Live: the instrument and the modulators all live on one Live track; add a second audio track set to Resampling to capture it.
Step 02

Build the audio path — then never touch it

Initialise and choose the engine
  • Shift + press the Type encoder to initialise a fresh patch — never sculpt on top of a factory preset.
  • Type encoder → Wavetable. It morphs harmonic content as the Wave knob moves, which is exactly what we want a modulator to drive. Harmo is the gentler alternative if Wavetable feels gritty.
  • Turn Paraphonic on — you will hold a chord, not a single note.
Starting knob positions
Oscillator + filter + amp — the fixed body
Wave~40 % — leaves room for a modulator to sweep it both ways
Timbre~55 %
Shape~50 %
Filter — Cutoff10 o'clock (dark) — the centre the modulator drifts around
Filter — Resonance9 o'clock — low; resonance + long reverb = whistling
ENV — Attacklong — the chord swells in, never clicks
ENV — Releaselong — notes fade rather than cut
Hold the chord — the fixed pitch world
  • Press Hold. Play the root A2, then add the fifth E3, then optionally the octave A3. All sustain hands-free.
  • This chord is the habitat's pitch world. The system will roam in brightness and timbre but never leave these notes. That is the single decision that keeps drift from becoming noise.
  • Listen for ten seconds with your hands off the synth. It should now be a static, sustained pad. Static is correct — the next step makes it move.
Decision check
  • With my hands completely off the synth, does a clean, sustained chord hold indefinitely?
  • Is the filter dark enough that there is somewhere brighter for the system to go?
  • Have I committed to this chord? Changing it later means re-recording everything.
Step 03

Wire the first modulator — LFO to cutoff

Set the LFO free, and slow
  • LFO Shape: Sine. Rate: unsynced — confirm it is not tempo-locked. This is non-negotiable; a synced LFO defeats the whole patch.
  • Set the rate to roughly one cycle every 14 seconds. On the MicroFreak this is near the slow end of the LFO range — turn the rate down until you can count "one-and-two-and…" through a full slow rise and fall over about fourteen seconds.
Route it, click by click
  • Rotate the Matrix encoder until the cursor sits on the LFO → Cutoff intersection.
  • Press to select the cell, turn to set the amount to ~15 (about 40 % of the way), press again to confirm.
  • Now the filter breathes on a fourteen-second clock. Watch the sound, hands off: it should slowly open and close, felt more than heard. If you hear an obvious wobble, halve the amount.
Same step in the other rigs
One slow LFO → cutoff, everywhere
VCV RackLFO 1 bipolar out → 8vert (~40 %) → VCF CUTOFF CV; LFO rate ~0.071 Hz
Max for LiveLFO A → map to Auto Filter Frequency; mode Hz at ~0.071 Hz; depth ~35 %
Decision check
  • Hands off: does brightness drift on a slow, roughly fourteen-second cycle?
  • Is the motion drift, not wobble? If you can tap along to it, it is too fast or too deep.
  • Is the LFO definitely free-running, not synced to tempo?
Step 04

Add the second motion — the cross-modulation

A second slow modulator on the timbre
  • Engage the Cycling Envelope in loop / Run mode — this turns it into a second free-running LFO. Set a slow rise and a slow fall so one full cycle takes about 23 seconds.
  • Matrix → Cycling Env → Wave, amount ~15. Confirm.
  • Now two motions run at 14 s and 23 s — unrelated periods. The brightness and the timbre drift on different clocks, so the combination never repeats. This is the engine described on page 03, now audible.
The true cross-modulation — modulate a rate
  • The deepest version of the trick is one modulator changing another modulator's speed. On the MicroFreak, route a slow source into the LFO's rate region so the fourteen-second cycle itself speeds up and slows down over minutes.
  • In VCV this is literal: LFO 2 out → 8vert (~25 %) → LFO 1 FM/rate input, LFO 2 at ~0.043 Hz (23 s).
  • In Max for Live: LFO C (period ~37 s) mapped to LFO A's Rate. Keep its depth shallow — a little rate-wobble goes a long way.
  • Keep the depth small. The point is that the period breathes, not that it lurches. If the motion ever reads as a seizure, this is the first depth to pull back.
Cyc Env · 23 s → wave LFO · 14 s → cutoff Held chord A–E → Audio out
Decision check
  • Do brightness and timbre clearly drift on different clocks?
  • Listen for 90 seconds, hands off: can you find two ten-second windows that sound identical? You should not.
  • Is anything thrashing? If so, the cross-modulation depth is too high — pull it back until motion reads as weather.
Step 05

Add a rare event — let a coin decide

Seasoning, not a part
  • A calm habitat can feel featureless. One occasional event — a brief swell or filter open — gives the ear something to wait for without breaking the drift.
  • VCV: a slow square (one pulse ~every 11 s) → Bernoulli Gate at ~25 % → an ADSR with long release → VCA or filter. It fires roughly once every 44 s, never on a schedule.
  • MicroFreak: there is no built-in coin, so emulate sparseness with a very slow second cycling-envelope shape that crests only occasionally, or add the rare event later in Ableton via an automated send bump (page 05, Recipe 4).
Keep it sparse
  • One event source only. Two coins make the listener start predicting, and prediction is the death of the effect.
  • Give every event a long recovery so it fades back into the wash with no hard edge.
Step 06

Set the boundaries the system cannot cross

Clamp the range, then process
  • The modulators must never reach harsh or vanish. The filter's centre plus the LFO depth defines the brightest and darkest the system can hit — set the cutoff centre and depth so the peak stays musical and the trough stays audible.
  • On the HABITAT Ableton track, build a fixed processing chain the system plays through. This is part of the habitat, not performance, so set it once:
Fixed processing on HABITAT — in this order
1 — EQ Eight HP40 Hz, 24 dB/oct — clears sub-rumble before the reverb
2 — EQ Eight LP shelf-3 dB above 7 kHz — caps brightness so a cutoff peak can't turn shrill
3 — Hybrid ReverbHall or Plate, decay 6–8 s, Dry/Wet 35 % — the room the habitat lives in
4 — Utility — Width120–130 % — the mono source gains controllable, mono-safe width
5 — Limiter (ceiling only)ceiling -1 dBFS as a safety net against an unexpected swell clipping the unattended take
  • EQ before reverb so you reverberate a clean signal, not rumble you then can't remove.
  • The limiter here is insurance, not loudness — it only catches a rare peak while you are out of the room.
Decision check
  • At the brightest the filter ever opens, is the sound still musical, not harsh?
  • At the darkest, is it still clearly audible, not gone?
  • Flip Utility to mono for ten seconds — does the habitat survive? If it thins out, pull Width back.
  • Can a sudden swell clip the master? The safety limiter should say no.
Step 07

The ten-minute unattended test

Run it, walk away, come back
  • Arm HABITAT, Monitor Auto. Position the playhead at 0:00. ⌘ ⇧ R to start recording.
  • Set a 10-minute timer and leave the room. Do not adjust anything. You are gathering evidence of the system's habits, not giving a performance.
  • On return, listen back to the full ten minutes from a different seat. Ask one question: did it stay interesting, or did it settle into a rut after two minutes?
Read the result, then fix the cause
Diagnosis → cause → fix
Settles into a rutmodulators too close in rate, or too fewwiden the mismatch (e.g. 14 & 23 & 37), or add a third
Runaway / seizurecross-modulation depth too highpull rate-modulation depth back until motion reads as drift
Goes silentamplitude modulated too deep, or release swamping attackraise the amp floor; shorten the modulation depth on level
Turns harshfilter peak exceeds the musical rangelower the cutoff centre or the LFO depth; tighten the LP shelf
Audibly loopstwo periods in a simple rationudge one period to a prime-ish, unrelated value
  • The goal: ten minutes where no thirty-second window sounds quite like another, yet the whole thing clearly came from one habitat.
  • Fix exactly one thing, then run the full ten minutes again. Resist tweaking mid-run — the test only means something if you stay out of the room.
A worked read of one ten-minute take
  • Here is what a healthy result actually sounds like, so you know what you are listening for. With LFO at 14 s on cutoff and the cycling envelope at 23 s on wave, the two cross only every couple of minutes. Around 2:10 both happen to crest together and the pad blooms bright and full — a peak you did not write. Around 4:40 the cutoff is opening while the wave is closing, and the timbre hollows out into something almost vocal.
  • Neither of those moments is on a schedule. Re-run the patch and the bloom lands at a different minute, because the phases start fresh. That unpredictability, inside a fixed chord and a fixed range, is the proof the system is alive.
  • Write two or three such timestamps down as you listen. They are your evidence for the "what the system decided" list, and they are the first thing page 07 asks you to produce.
  • If, instead, you hear the same bloom at 2:10, 4:20, and 6:30 — a steady two-minute pulse — your periods are too close to a ratio. That regularity is the rut, and the fix is to widen one period, not to add busier modulation.
Decision check
  • Did the ten minutes hold attention without me touching anything?
  • Can I find two identical windows? (I should not.)
  • Could I have predicted minute six from minute one? (I should not be able to.)
  • Did at least one moment surprise me — a bloom or a hollowing I did not design?
Step 08

Capture the keeper take

Record the definitive ten minutes
  • Once a tuning passes the test, save the patch: on the MicroFreak press Save, write it to an empty slot as B6 HABITAT. The system that worked must survive the session.
  • Record a fresh, clean ten-minute take with the tuned settings — the test take usually has a false start or a chair creak in it.
  • Hands off the whole time. Any knob you touch is no longer the system playing — it is you playing. That distinction is the entire exercise.
  • If you want a longer field recording, record fifteen or twenty minutes; you will curate the best ten on page 07's terms.
Light curation — the only place your hand returns
  • You may choose where the ten minutes start and end within a longer recording. That is curation, and it is allowed — Eno chose which loops to release too.
  • Draw a slow fade-in over the first 15–20 s and a slow fade-out over the last 30–45 s using the clip fade handles. Nothing else gets edited — no comping, no rearranging.
  • Make any boundary cut at a quiet moment in the drift, so the entrance and exit feel like opening and closing a door on a room that was already alive.

Archive the system in three layers

  • Save the patch on the hardware (B6 HABITAT), or save the VCV .vcv patch file, or the Live set with its Max devices. The habitat must be re-runnable.
  • Write the two lists from page 03 into self_player_b6_002_notes.txt: what I decided (chord, range, rates, depths) and what the system decided (which peaks landed where). Page 07 needs both.
Decision check
  • If I reload the patch and run it again, will it produce a different ten minutes from the same habitat? (It should — same rules, new combination.)
  • Is my "what I decided" list non-empty? (If empty, I abdicated — the rates were random.)
  • Is my "what the system decided" list non-empty? (If empty, I built a slow performance, not a self-player.)
  • If you used VCV or Max for Live: every step above maps directly. The audio path is the chord/oscillator chain; the control path is the unsynced LFOs; the test, the capture, and the two-list archive are identical. Only the cabling changed.
  • The Strega guest: if you sampled the Strega's self-feedback into the Digitakt II, lay one quiet pass underneath the captured habitat as a second organism — never louder than a whisper, never a second lead.
