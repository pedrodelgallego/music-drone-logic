Lab Note B6-002
Self-playing Patches · 02 — Listen

Hearing a system play itself

Before you patch a single cable, train the ear. These three recordings are not performances — they are habitats left running. Don't copy them; listen for the mechanism: what stays fixed, what drifts, and how nothing ever exactly repeats. Each links to a full replication lesson in Block 6.

Three studies in systems left running

Brian Eno — "2/1", Music for Airports

  • Tape loops of different lengths drift in and out of phase
  • No note is composed; the system of overlapping cycles is
  • Listen for the combination never repeating inside the window

Keith Fullerton Whitman — "Generator 1"

  • An analogue generative patch left to run and recorded whole
  • Cross-modulating voltages, no two minutes the same
  • Hear the human's absence — the patch is the performer

Roland Kayn — Tektra

  • Vast "cybernetic music": feedback networks left to self-organise
  • The composer sets the network, then the network composes
  • Listen for motion that is clearly governed yet never schedulable

A structured way to listen

Pass 01

Find what never changes

The fixed boundaries
  • Play the first two minutes twice. On the second pass, ignore everything that moves and name what is always present: a chord, a register, a timbral family.
  • That fixed body is the habitat's walls. The drift only reads as drift because it happens inside something stable.
  • In all three references the constant is harmonic and tonal — the system roams within a fixed pitch world, it does not wander out of key.
Pass 02

Find the slow motions

What drifts, and how fast
  • Pick one timbre and follow only it for a minute. Time roughly how long one full swell-and-return takes — that is one modulator's period.
  • Now follow a different timbre. Its period will be different. The two never line up, which is exactly why the combination keeps changing.
  • Count how few independent motions there really are. It is almost always two or three slow cycles producing an illusion of endless variety.
Pass 03

Test for repetition

Looking for the seam
  • Try to find two moments that sound identical. In a true self-player you cannot — the cycles are at unrelated rates, so the exact combination never recurs in the listening window.
  • If you do hear an obvious loop point, you are hearing a fixed arrangement, not a system. Note the difference: that is the line this Lab Note is built around.
  • Write down, in plain words, the smallest rule set you'd need to grow 30 seconds of the piece. That sentence is your brief for the build.
  • The originality constraint: these recordings are ear-training, not templates. Take the mechanism — fixed habitat, unsynced slow motions, no exact repetition — and grow something entirely yours. The patch you build in this Lab Note must not reproduce any reference's pitch material, loop, or arrangement.
