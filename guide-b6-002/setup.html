Lab Note B6-002
Self-playing Patches · 04 — Setup

Three habitats, one idea

The same self-playing principle works in free software, in owned hardware, or entirely in the box. Pick the rig you have. Each section below gives the audio path, the control path, and the recording route — so the build page can assume the wiring is done.

Keep two paths separate in your head

Every habitat has an audio path (oscillator → filter → amp → out) you set once and never touch again, and a control path (the LFOs and gates) that is the living part. The whole craft is in the control path. Wire the audio path, walk away from it, and spend your attention on the modulators.

Audio path VCO → VCF → VCA → out (set once) Control path LFOs + gates → the living part

Rig A — VCV Rack (free, recommended first)

Rig A — free

VCV Rack as habitat

Modules to place (all stock VCV)
  • VCO-1 → VCF → VCA-1 → Audio-8 output. That is the entire audio path; once it makes a tone you never touch it again.
  • Three LFO modules. Set each to its bipolar low-frequency output, and switch the VCO down into low-frequency / drone territory or hold a fixed pitch with a Quantizer + fixed CV.
  • One Bernoulli Gate (a coin) fed by a slow LFO square — for rare retriggered events.
  • One ADSR + 8vert (attenuverter) to scale CV depths precisely.
Patch the control path
VCV cabling — the living part
LFO 1 out→ VCF CUTOFF CV (via 8vert, ~40 %)
LFO 2 out→ LFO 1 FM / rate input (via 8vert, ~25 %) — the cross-modulation
LFO 3 out→ VCO timbre / wave or VCA level (via 8vert, ~20 %)
Slow square → BernoulliBernoulli A out → ADSR gate → VCA — rare swells
  • Keep audio cables one colour habit and CV cables another. VCV makes the split visual — use it.
  • Recording: drop a Recorder module (or VCV's built-in audio recorder) on the master, or route Audio-8 into your DAW. This is what you will leave running for ten minutes.
Why VCV first
  • Free and unlimited — build a 40-module habitat you could never afford in hardware.
  • Everything transfers directly to the MicroFreak matrix and any Eurorack rig later. The concepts are identical; only the cabling changes.

Rig B — MicroFreak matrix (owned hardware)

Rig B — owned

The MicroFreak matrix as habitat

Audio path — set once
  • Shift + press the Type encoder to initialise a fresh patch. Engine Wavetable or Harmo for a smooth, controllable timbre.
  • Press Hold on. Play a chord — root and fifth, or a triad. It sustains hands-free; this is your fixed pitch world.
  • Filter cutoff to around 10 o'clock (dark), resonance low. Long attack and release on the amp envelope so nothing clicks.
  • MicroFreak Audio Out → MOTU M4 Input 1 with one 1/4" TS cable. This is the recording feed.
Control path — wire the matrix, click by click
Mod matrix routes — unsynced, slow
LFO → Cutoffamount ~15. LFO Shape Sine, Rate unsynced, ~1 cycle / 14 s
Cycling Env → Waveamount ~15. Cycling Env in loop / Run mode, slow rise+fall, ~1 cycle / 23 s
Cycling Env → LFO areaoptional ~10 — feeds a second slow motion into the timbre
Pressure → Timbreleft at zero — no human hand on it. The point is unattended motion.
  • To set a route: rotate the Matrix encoder to the intersection, press to select, turn to set the amount, press again to confirm.
  • Two modulators at unrelated periods (14 s vs 23 s) is the same trick as Rig A. If you can clearly hear a wobble, halve the depth.
  • Confirm the LFO rate is not tempo-synced — for a self-player it must free-run.
Optional wildcard — Make Noise Strega
  • The Strega's internal feedback and "no two patches the same" character makes it an unstable organism on its own. Patch its outputs back into its inputs and it self-modulates.
  • Treat it as a guest creature in the terrarium: sample a few minutes of it into the Digitakt II rather than trying to tame it live.
  • Keep it quiet underneath the MicroFreak habitat — a second organism, not a second lead.

Rig C — Live + Max for Live (in the box)

Rig C — in the box

Live + Max for Live habitat

Audio path — set once
  • One instrument track with Wavetable or Operator, a sustained chord drawn as a long MIDI clip (or held via a single long note + sustain). Add an Auto Filter after it — this is the parameter the system will move most.
  • An audio track set to record the instrument track's output — the field recorder for the ten-minute capture.
Control path — modulators with no host hand
Max for Live LFO devices — free-running, not synced
LFO Amap to Auto Filter Frequency. Mode Hz (free), ~0.07 Hz (~14 s), depth ~35 %
LFO Bmap to a reverb send or the wavetable position. Mode Hz, ~0.043 Hz (~23 s), depth ~30 %
LFO Cmap to LFO A's own Rate (via a mapped macro) — the cross-modulation. ~0.027 Hz (~37 s), shallow
  • The key switch on every LFO is free-running in Hz, not sync. Sync would lock the periods into musical ratios and defeat the whole exercise.
  • Mapping an LFO to another LFO's Rate is the in-the-box version of patching CV into a rate input. Use the Max LFO's Map button, target LFO A's Rate.
  • Arm the audio track and let it record the habitat for as long as you'll later want to curate. The Live set is the terrarium; the recording is the field footage.

Free code paths

  • The unattended generator is a few lines of SuperCollider — { SinOsc.kr } modulators driving a filter and an oscillator at unrelated rates.
  • Or a Pure Data patch of [osc~] LFOs cross-modulating one another's frequency inlets.
  • Or VCV Rack, above. None of these cost anything to try, and all of them implement the identical idea: unsynced slow motions, fixed pitch world, recorded hands-off.
Before you build

Whichever rig you chose, confirm one thing: with your hands completely off the instrument, does it already move? If it sits still, the control path isn't wired yet. Fix that here, not on the build page.
